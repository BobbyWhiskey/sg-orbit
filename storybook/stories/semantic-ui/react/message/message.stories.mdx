# Message

A message displays information that explains nearby content.

## Installation

<details><summary >install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a standalone component:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
```
</details>

<details><summary >then import what you need</summary>

```javascript
    import { Message } from "semantic-ui-react";
```
</details>

## Usage

### Default

A default message.

<Preview>
    <Story name="default">
        <Message>
            Easily manage external sharing links and set-up automatic access reviews for entrusted group owners.
        </Message>
    </Story>
</Preview>

### Header

A message can have an header.

<Preview>
    <Story name="header">
        <Message>
            <Message.Header>Setup a security policy for external sharing</Message.Header>
            <p>
                Easily manage external sharing links and set-up automatic access reviews for entrusted group owners.
            </p>
        </Message>
    </Story>
</Preview>

### List

A message can contains a formatted list.

<Preview>
    <Story name="list">
        <Message>
            <Message.Header>New Site Features</Message.Header>
            <Message.List>
                <Message.Item>You can now have cover images on blog pages</Message.Item>
                <Message.Item>Drafts will now auto-save while writing</Message.Item>
            </Message.List>
        </Message>
    </Story>
</Preview>

### Dismissible

A message can be dismissed.

<p>
    <span style={{ display: "inline-block", lineHeight: 1, verticalAlign: "middle" }}><InfoIcon className="w6 h6" /></span> <span style={{ fontStyle: "italic" }}>Dismissible blocks do not automatically close. Use the <code>onDismiss</code> callback to remove the message.</span>
</p>

<Preview>
    <Story name="dismissible">
        <Message onDismiss={noop}>
            <Message.Header>New Site Features</Message.Header>
            <Message.List>
                <Message.Item>You can now have cover images on blog pages</Message.Item>
                <Message.Item>Drafts will now auto-save while writing</Message.Item>
            </Message.List>
        </Message>
    </Story>
</Preview>
