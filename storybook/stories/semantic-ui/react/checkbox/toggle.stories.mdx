# Toggle

A toggle show an on or off choice.

## Installation

<Installation imports={<Imports />} />

## Usage

### Labelled

A toggle can have a label.

<Preview>
    <Story name="labelled">
        <Checkbox toggle label="Milky Way"/>
    </Story>
</Preview>

### On

A toggle can be on.

<Preview>
    <Story name="on">
        <Checkbox toggle defaultChecked label="Milky Way"/>
    </Story>
</Preview>

### Disabled

A toggle can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox toggle disabled label="Milky Way"/>
    </Story>
</Preview>

### No label

A toggle doesn't require a label.

<Preview>
    <Story name="no label">
        <Checkbox toggle />
    </Story>
</Preview>
