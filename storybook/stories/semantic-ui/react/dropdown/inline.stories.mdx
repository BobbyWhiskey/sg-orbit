# Inline Dropdown

An inlined dropdown can appear inline with other content.

## Installation

<Installation imports={<Imports />} />

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="/?path=/docs/components-dropdown-standard--default-story">standard Dropdown</a> component.
</WarningMessage>

### Default

A default inlined dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            placeholder="select friend"
            inline
            options={[
                {
                    key: "Jenny Hess",
                    text: "Jenny Hess",
                    value: "Jenny Hess"
                },
                {
                    key: "Elliot Fu",
                    text: "Elliot Fu",
                    value: "Elliot Fu"
                }
            ]}
        />
    </Story>
</Preview>

### In a block

An inlined dropdown can be embedded in a block.

<Preview>
    <Story name="in a block">
        <div>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Pellentesque vestibulum et<br /> et lacus at euismod <Dropdown
            placeholder="select friend"
            inline
            options={[
                {
                    key: "Jenny Hess",
                    text: "Jenny Hess",
                    value: "Jenny Hess"
                },
                {
                    key: "Elliot Fu",
                    text: "Elliot Fu",
                    value: "Elliot Fu"
                }
            ]}
        /> elementum viverra maximus.</div>
    </Story>
</Preview>
