# Inline Dropdown

An inlined dropdown can appear inline with other content.

## Installation

<PackageInstallation />

<details><summary >then import what you need</summary>

```javascript
    import { Dropdown } from "semantic-ui-react";
```
</details>

## Usage

<Warning>
    This component have the same feature set as the <a href="/?path=/docs/components-dropdown-standard--default-story">standard Dropdown</a> component.
</Warning>

### Default

A default inlined dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            placeholder="select friend"

            options={[
                {

</Preview>
