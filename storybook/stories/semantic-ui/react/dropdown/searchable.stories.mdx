# Searchable Dropdown

A dropdown can allow a user to search through a large list of choices.

## Installation

<details><summary >install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a standalone component:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
```
</details>

<details><summary >then import what you need</summary>

```javascript
    import { Dropdown } from "semantic-ui-react";
```
</details>

## Usage

<Warning>
    This component have the same feature set as the <a href="/?path=/docs/components-dropdown-standard--default-story">Dropdown</a> component.
</Warning>
