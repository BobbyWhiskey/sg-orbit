# Dropdown

A dropdown can be used to select between choices.

## Installation

<Installation imports={<Imports />} />

## Usage

### Default

A default dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            selection
            options={[

                    key: "Male",

                {
                    key: "Female",

                    value: "Female"

</Preview>
