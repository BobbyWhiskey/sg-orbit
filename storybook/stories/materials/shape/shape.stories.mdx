## Radiuses

An element can have a border radius modifying its shape.

| Scale | Value | Class   |
| :---- | :---- | :----   |
| 1     | 2px   | br1     |
| 2     | 4px   | br2     |
| 3     | 8px   | br3     |
| 4     | 16px  | br4     |
| pill  | 999px | br-pill |
| 100   | 100%  | br-100  |
| 0     | 0     | br0     |
