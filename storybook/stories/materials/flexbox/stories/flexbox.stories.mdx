# Flexbox

Flex can be used to achieve powerful horizontal or vertical layouts without JavaScript. Before using these utilities, you should be familiar with CSS3 Flexible Box specifications. If you are not, check out this [CSS-Tricks guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/).

You will find most of the documentation for flexbox utilities on [Tachyons website](http://tachyons.io/docs/layout/flexbox/). For an exhaustive list of utilities classes for flexbox **including** our customizations, please refer to the [Tachyons section](#tachyons) below.
