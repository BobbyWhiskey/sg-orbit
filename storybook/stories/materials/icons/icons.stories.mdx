# Icons

Icons includes their padding, they generally are available in two sizes, 24px and 32px. When only one size is provided they are considered universal. Default size is 32px, when available, adding 24 at the end of an icon will bring the 24x24 version of this icon.

## Installation

<details><summary >Install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:

```bash
npm install @orbit-ui/icons
```
</details>

## Usage

<details><summary >as a React component</summary>

```jsx
import { IconName } from "@orbit-ui/icons";
```
</details>

<details><summary >as an image</summary>

```jsx
import IconName from "@orbit-ui/icons/icon-name.svg";

<img src={IconName} />
```
</details>

<details><summary >as a background image</summary>

```css
.name-icon: {
    background: url("~@orbit-ui/icons/icon-name.svg");
}
```
</details>
