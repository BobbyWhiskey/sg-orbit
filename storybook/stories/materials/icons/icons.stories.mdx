# Icons

Icons includes their padding, they generally are available in two sizes, 24px and 32px. When only one size is provided they are considered universal. Default size is 32px, when available, adding 24 at the end of an icon will bring the 24x24 version of this icon.

## Installation

<Installation />

## Usage

Icons does not have a default dimensions to favour reusability. This means that you must specify dimensions either with [Tachyons](/?path=/docs/introduction-tachyons--page) dimensions classes (e.g. `w6 h6` for a 24x24 icon) or straight CSS.

#### Don't forget to Specify a size.

```html
<AddIcon24 className="w6 h6"/>
```
