# Shadows

Shadows can be used to elevate an element.

You will find most of the documentation for shadows utilities on [Tachyons website](http://tachyons.io/docs/themes/box-shadow/). Keep in mind though that all the values have been costumized to match Orbit Design. For an exhaustive list of utilities classes for shadows **including our customizations**, please refer to the [Tachyons section](#tachyons) below.
