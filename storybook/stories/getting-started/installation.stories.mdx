# Installation

Multiple [npm packages](?path=/docs/getting-started-packages--page) compose Orbit UI. Based on which type of application you're building you'll choose a different composition.

## React bundle

For a React application, the easiest way to use Orbit UI is by installing the [@orbit-ui/react](https://www.npmjs.com/package/@orbit-ui/react) bundle. The bundle includes all the [core](?path=/docs/getting-started-packages--page#core) and [React components](?path=/docs/getting-started-packages--page#react-components) packages.

### Installation

You can install the bundle in 3 easy steps.

#### Install packages

```bash dark
npm install @orbit-ui/react react-spring moment lodash
```

#### Import components

```javascript dark
import { Button, DateRangePicker, MultiSelect, ... } from "@orbit-ui/react-components";
import { Message, ... } from "semantic-ui-react";
import { ClearIcon, ... } from "@orbit-ui/icons";
```

#### Import styles

```css dark
@import "~@orbit-ui/react/[(apricot|overcast|desktop).css]";
@import "~@orbit-ui/semantic-ui-theme";
```

## Custom installation

If your app doesn't use React or you prefer to hand-pick a few [packages](?path=/docs/getting-started-packages--page), this is also possible.

Strict *peerDependencies* are defined in every packages to ensure that dependent packages are installed together. This means that if a required package is missing during the installation, a warning will be output by your package manager.

### Core packages

Before installing any [React components packages](?path=/docs/getting-started-packages--page#react-components), you must install the [core packages](?path=/docs/getting-started-packages--page#core). You can pick and choose the *core packages* to install but we do recommend installing all of them.

#### Install all core packages

```bash dark
npm install @orbit-ui/fonts @orbit-ui/css-normalize @orbit-ui/icons @orbit-ui/foundation @orbit-ui/tachyons
```

#### Import styles

```css dark
@import "~@orbit-ui/fonts";
@import "~@orbit-ui/css-normalize";
@import "~@orbit-ui/foundation[/(apricot|overcast|desktop).css]";
@import "~@orbit-ui/tachyons[/(apricot|overcast|desktop|legacy).css]";
```

### React component packages

Once the *core packages* are installed, you can optionally add *React component packages*. You can either pick and choose the component packages to install or install all the components and leverage [Webpack treeshaking](https://webpack.js.org/guides/tree-shaking/).

#### Install all components

```bash dark
npm install @orbit-ui/react-components @orbit-ui/semantic-ui-theme semantic-ui-react react-spring styled-jsx moment lodash
```

#### Hand-pick a few components to install

```bash dark
npm install @orbit-ui/react-* @orbit-ui/semantic-ui-theme semantic-ui-react styled-jsx
```

#### Import components

```javascript dark
import { Button, DateRangePicker, MultiSelect, ... } from "@orbit-ui/react-components";
import { SingleDatePicker } from "@orbit-ui/react-date-pickers";
import { Message, ... } from "semantic-ui-react";
```

### Semantic UI theme

When you're done importing components, you must include Semantic UI theme to style your components. You can either pick and choose the specific modules that you need or import the whole theme.

#### Pick and choose specific modules

```css dark
@import "~@orbit-ui/semantic-ui-theme/button.css";
@import "~@orbit-ui/semantic-ui-theme/checkbox.css";
```

#### Import the whole theme

```css dark
@import "~@orbit-ui/semantic-ui-theme";
```

## Icons

Some parts of [Orbit icons](?path=/docs/materials-icons--page) depends on [SVGR](https://github.com/gregberge/svgr) to transform SVG icons files into React components. If you use Orbit icons React components or any Orbit React components, chances are you depend on SVGR. To use SVGR, your project babel config must configure the [babel-plugin-named-asset-import](https://www.npmjs.com/package/babel-plugin-named-asset-import) plugin:

```js dark
{
    plugins: [
        [
            "babel-plugin-named-asset-import",
            {
                loaderMap: {
                    svg: {
                        "ReactComponent": "@svgr/webpack?-svgo,+ref![path]"
                    }
                }
            }
        ]
    ]
}
```
