# Foundation

The foundation is the core of Orbit. It contains [CSS variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) that store the visual design attributes of the design system. We use those variables across Orbit instead of hard-coded values in order to maintain a scalable and consistent visual system for UI development.

Once [installed](?path=/docs/getting-started-installation--page), the default values can be [customized to your needs](#override-variables). Please be aware that customization would no longuer benefinits from updates and could break.

## Use variables

A foundation CSS variable can be accessed in a CSS class:

```css dark
.button {
    color: var(----primary-50);
}
```

In a `style` attribute:

```html dark
<button style="color: var(--primary-50)" type="button">
    Click me!
</button>
```

Or in JavaScript (The computed variable value is available as a string):

```javascript dark
window.getComputedStyle(document.documentElement).getPropertyValue("--primary-50");
```

## Override variables

The foundation CSS variables can be overrided by simply redeclaring the variables in a `:root` CSS class:

```css dark
:root {
    --primary-50:  #c63a0c;
    --apricot-50: #c63a0c;
    --overcast-50: #c63a0c;
    --desktop-50: #c63a0c;
}
```

If you need to use a dynamic value or to scope an update to a specific element you can override a variable with a `style` attribute:

```html dark
<button style="--primary-50: #c63a0c" type="button">
    Click me!
</button>
```

At last resort, if all the techniques described above doesn't work for you, you can override a variable with JavaScript. However, we strongly encourage using a CSS class or a `style` attribute.

```javascript dark
document.body.style.setProperty("--primary-50", "#c63a0c");
```
