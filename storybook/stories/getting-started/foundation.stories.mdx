# Foundation

Once [installed](?path=/docs/getting-started-installation--page), Orbit CSS foundation can be customized to your needs. Please be aware that customization would no longuer benefinits from updates and could break.

## Override default variables

The foundation default variables can be overrided by simply redeclaring the variables in a `:root` CSS class:

To override the <InlineGithubLink path="/packages/foundation/src/core">core variables</InlineGithubLink>:

```css dark
:root {
    --primary-50:  #c63a0c;
}
```

To override brand specific variables for <InlineGithubLink path="/packages/foundation/src/apricot.css">Apricot</InlineGithubLink>, <InlineGithubLink path="/packages/foundation/src/overcast.css">Overcast</InlineGithubLink> or <InlineGithubLink path="/packages/foundation/src/desktop.css">Desktop</InlineGithubLink>:

```css dark
:root {
    --apricot-50: #c63a0c;
    --overcast-50: #c63a0c;
    --desktop-50: #c63a0c;
}
```
