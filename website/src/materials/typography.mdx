# Typography

## Font Sizes

Font sizes are based on a scale of 9

| Scale | Value           |
| :---- | :-------------- |
| 1     | 3rem (48px)     |
| 2     | 2.5rem (40px)   |
| 3     | 2rem (32px)     |
| 4     | 1.5rem (24px)   |
| 5     | 1.25rem (20px)  |
| 6     | 1.125rem (18px) |
| 7     | 1rem (16px)     |
| 8     | 0.875rem (14px) |
| 9     | 0.75rem (12px)  |
