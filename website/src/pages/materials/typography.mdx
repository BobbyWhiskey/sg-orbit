# Typography

## Font Size Scale

Font sizes are based on a scale of 9

| Scale | Value           |
| :---- | :-------------- |
| 1     | 3rem (48px)     |
| 2     | 2.5rem (40px)   |
| 3     | 2rem (32px)     |
| 4     | 1.5rem (24px)   |
| 5     | 1.25rem (20px)  |
| 6     | 1.125rem (18px) |
| 7     | 1rem (16px)     |
| 8     | 0.875rem (14px) |
| 9     | 0.75rem (12px)  |

## Line Heights

There is 6 line heights available to you, these are based on a scale principle.

## Line Height Scale

| :---- | :----------- |
| 1     | 1            |
| 2     | 1.2          |
| 3     | 1.25         |
| 4     | 1.33333      |
| 5     | 1.4285714286 |
| 6     | 1.5          |
