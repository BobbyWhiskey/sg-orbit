# Tag

A tag represent an interactive keyword that help label, organize and categorize content. Tags are meant to be displayed in a [TagsPicker](?path=/story/components-tagspicker--default-story).

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Tag", component: InnerTag }
    ]}
/>

## Slots

A tag component support the following [slots](?path=/docs/getting-started-slots--page): `"text"`, `"icon"`, `"dot"`, `"counter"`, `"button"`.

## Usage

### Default

A default tag.

<Preview>
    <Story name="default">
        <Tag>Falcon 9</Tag>
    </Story>
</Preview>

### Button

A tag can contain a [button](?path=/docs/components-button--default-story).

<Preview>
    <Story name="button">
        <Tag>
            <Text>Falcon 9</Text>
            <IconButton aria-label="Clear value">
                <CrossIcon />
            </IconButton>
        </Tag>
    </Story>
</Preview>

### Icon

A tag can contain [icons](?path=/docs/materials-icons--page).

<Preview>
    <Story name="icon">
        <Inline>
            <Tag>
                <CheckCircleIcon />
                <Text>Falcon 9</Text>
            </Tag>
            <Tag>
                <IconGroup>
                    <CheckCircleIcon /><CheckCircleIcon />
                </IconGroup>
                <Text>Falcon 9</Text>
            </Tag>
        </Inline>
    </Story>
</Preview>

### Dot

A tag can contain a [dot](?path=/docs/components-dot--default-story).

<Preview>
    <Story name="dot">
        <Tag>
            <Dot color="primary-500" />
            <Text>Falcon 9</Text>
        </Tag>
    </Story>
</Preview>

### Counter

A tag can contain a [counter](?path=/docs/components-counter--default-story).

<Preview>
    <Story name="counter">
        <Tag>
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Preview>

### Size

A tag can vary in size.

<Preview>
    <Story name="size">
        <Inline align="end">
            <Tag size="small">Falcon 9</Tag>
            <Tag>Falcon 9</Tag>
            <Tag size="large">Falcon 9</Tag>
        </Inline>
    </Story>
</Preview>

### Fluid

A tag can take the width of its container.

<Preview>
    <Story name="fluid">
        <Tag fluid>
            <CheckCircleIcon />
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Preview>

### Outline

A tag can have a subtle appearance.

<Preview>
    <Story name="outline">
        <Tag variant="outline">Falcon 9</Tag>
    </Story>
</Preview>

### Custom colors

Custom colors can be applied to a tag.

<Preview>
    <Story name="custom-color">
        <Tag
            style={{
                backgroundColor: "#FFF6E7",
                color: "var(--marine-500)",
                border: "1px solid #FEE9C3"
            }}
        >
            Falcon 9
        </Tag>
    </Story>
</Preview>
