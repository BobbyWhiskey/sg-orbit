# Tag

A tag represent an interactive keyword that help label, organize and categorize content. Tags are meant to be displayed in a [TagsPicker](?path=/story/components-tagspicker--default-story).

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Tag", component: InnerTag }
    ]}
/>

## Usage

### Default

A default tag.

<Preview>
    <Story name="default">
        <Tag>Falcon 9</Tag>
    </Story>
</Preview>

### Button

A tag can contain a [button](?path=/docs/components-button--default-story).

<Preview>
    <Story name="button">
        <Tag button={<Button icon={<CloseIcon />} />}>
            Falcon 9
        </Tag>
    </Story>
</Preview>

### Icon

A tag can contain a left or right [icon](?path=/docs/materials-icons--page).

<Preview>
    <Story name="icon">
        <div className="flex items-space-between">
            <Tag icon={<InfoIcon />}>
                Falcon 9
            </Tag>
            <Tag icon={<CheckIcon />} iconPosition="right">
                Falcon 9
            </Tag>
        </div>
    </Story>
</Preview>

### Dot

A tag can contain a [dot badge](?path=/docs/components-badge--dot).

<Preview>
    <Story name="dot">
        <div className="flex items-space-between">
            <Tag dot={<Badge dot className="bg-primary-500" />}>
                Falcon 9
            </Tag>
        </div>
    </Story>
</Preview>

### Basic

A tag can have a subtle appearance.

<Preview>
    <Story name="basic">
        <Tag basic>Falcon 9</Tag>
    </Story>
</Preview>

### Size

A tag can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Tag size="small">Falcon 9</Tag>
            <Tag>Falcon 9</Tag>
            <Tag size="large">Falcon 9</Tag>
        </div>
    </Story>
</Preview>

### Compact

A tag can reduce its padding to fit into tighter spaces.

<Preview>
    <Story name="compact">
        <Tag compact>Falcon 9</Tag>
    </Story>
</Preview>

### Custom colors

Custom colors can be applied to a tag.

<Preview>
    <Story name="custom-color">
        <Tag
            style={{
                backgroundColor: "#FFF6E7",
                color: "var(--marine-500)",
                border: "1px solid #FEE9C3"
            }}
        >
            Falcon 9
        </Tag>
    </Story>
</Preview>
