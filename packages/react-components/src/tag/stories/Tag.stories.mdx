# Tag

A tag represent an interactive keyword that help label, organize and categorize content. Tags are meant to be displayed in a [TagsPicker](?path=/story/components-tagspicker--default-story).

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Tag", component: InnerTag }
    ]}
/>

## Slots

Support `"text"`, `"icon"`, `"dot"` and `"counter"` [slots](?path=/docs/getting-started-slots--page).

## Usage

### Default

A default tag.

<Preview>
    <Story name="default">
        <Tag>Falcon 9</Tag>
    </Story>
</Preview>

### Icon

A tag can contain [icons](?path=/docs/materials-icons--page).

<Preview>
    <Story name="icon">
        <Tag>
            <CheckCircleIcon />
            <Text>Falcon 9</Text>
        </Tag>
    </Story>
</Preview>

### Dot

A tag can contain a [dot](?path=/docs/components-dot--default-story).

<Preview>
    <Story name="dot">
        <Tag>
            <Dot color="primary-500" />
            <Text>Falcon 9</Text>
        </Tag>
    </Story>
</Preview>

### Counter

A tag can contain a [counter](?path=/docs/components-counter--default-story).

<Preview>
    <Story name="counter">
        <Tag>
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Preview>

### Removable

A tag can be removable by specifying an `onRemove` handler.

<Preview>
    <Story name="removable">
        <Tag onRemove={() => {}}>Falcon 9</Tag>
    </Story>
</Preview>

### Size

A tag can vary in size.

<Preview>
    <Story name="size">
        <Inline verticalAlign="end">
            <Tag size="sm">Falcon 9</Tag>
            <Tag>Falcon 9</Tag>
            <Tag size="lg">Falcon 9</Tag>
        </Inline>
    </Story>
</Preview>

### Fluid

A tag can take the width of its container.

<Preview>
    <Story name="fluid">
        <Tag fluid>
            <CheckCircleIcon />
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Preview>

### Outline

A tag can have a subtle appearance.

<Preview>
    <Story name="outline">
        <Tag variant="outline">Falcon 9</Tag>
    </Story>
</Preview>

### Custom colors

Custom colors can be applied to a tag.

<Preview>
    <Story name="custom-color">
        <Tag
            style={{
                backgroundColor: "#FFF6E7",
                color: "var(--marine-500)",
                border: "1px solid #FEE9C3"
            }}
        >
            Falcon 9
        </Tag>
    </Story>
</Preview>
