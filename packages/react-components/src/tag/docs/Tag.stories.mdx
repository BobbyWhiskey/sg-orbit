# Tag

<Tagline>A tag represent an interactive keyword that help label, organize and categorize content. Tags are meant to be displayed in a [TagsPicker](?path=/story/tagspicker--default-story).</Tagline>

<ComponentInfo
    usage={"import { Tag } from \"@orbit-ui/react-components\""}
    githubPath="/packages/react-components/src/tag/src"
/>

## Usage

### Default

A default tag.

<Canvas>
    <Story name="default">
        <Tag>Falcon 9</Tag>
    </Story>
</Canvas>

### Icon

A tag can contain [icons](?path=/docs/icon-gallery--page).

<Canvas>
    <Story name="icon">
        <Tag>
            <CheckCircleIcon />
            <Text>Falcon 9</Text>
        </Tag>
    </Story>
</Canvas>

### Dot

A tag can contain a [dot](?path=/docs/dot--default-story).

<Canvas>
    <Story name="dot">
        <Tag>
            <Dot color="primary-500" />
            <Text>Falcon 9</Text>
        </Tag>
    </Story>
</Canvas>

### Counter

A tag can contain a [counter](?path=/docs/counter--default-story).

<Canvas>
    <Story name="counter">
        <Tag>
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Canvas>

### Removable

A tag can be removable by specifying an `onRemove` handler.

<Canvas>
    <Story name="removable">
        <Tag onRemove={() => {}}>Falcon 9</Tag>
    </Story>
</Canvas>

### Size

A tag can vary in size.

<Canvas>
    <Story name="size">
        <Inline verticalAlign="center">
            <Tag size="sm">Falcon 9</Tag>
            <Tag>Falcon 9</Tag>
        </Inline>
    </Story>
</Canvas>

### Fluid

A tag can take the width of its container.

<Canvas>
    <Story name="fluid">
        <Tag fluid>
            <CheckCircleIcon />
            <Text>Falcon 9</Text>
            <Counter>6</Counter>
        </Tag>
    </Story>
</Canvas>

### Outline

A tag can have a subtle appearance.

<Canvas>
    <Story name="outline">
        <Tag variant="outline">Falcon 9</Tag>
    </Story>
</Canvas>

### Custom colors

Custom colors can be applied to a tag.

<Canvas>
    <Story name="custom-color">
        <Tag
            style={{
                backgroundColor: "#FFF6E7",
                color: "var(--marine-500)",
                border: "1px solid #FEE9C3"
            }}
        >
            Falcon 9
        </Tag>
    </Story>
</Canvas>
