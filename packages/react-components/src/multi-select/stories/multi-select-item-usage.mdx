A multi select component requires a set of items to display. Those items can be specified with the `items` property.

To create an item, use the `multiSelectItem(text, value, category?, additionalProps?)` function:

```javascript dark
const apollo = multiSelectItem("Apollo Program", "apollo");
```

<details><summary >optional category</summary>

An optional `category` can be specified to group a set of items under a labelled category:

```javascript dark
const apollo = multiSelectItem("Apollo Program", "apollo", "1970");
const skylab = multiSelectItem("Skylab", "skylab", "1970");
```
</details>

<details><summary >specify optional data</summary>

Additional properties can be specified to provide additional information to any callbacks e.g. `onValuesChange` or `itemRenderer`.

```javascript dark
const apollo = multiSelectItem("Apollo Program", "apollo", null, { backendId: 123 });
```

Then in any callbacks:

```jsx dark
<MultiSelect
    items={[groupCreated]}
    onValuesChange={(event, values) => {
        values.forEach(x => console.log(x.backendId));
    }}
/>
```

Or in a custom result renderer:

```jsx dark
<MultiSelect
    items={[groupCreated]}
    selectedItemsComponent={
        <MultiSelect.SelectedItems
            itemRenderer={({ text, value, group, backendId }) => {
                return <div>{backendId}</div>
            }}
        />
    }
/>
```
</details>
