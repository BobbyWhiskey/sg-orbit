# Toolbar

A toolbar is a container for grouping a set of controls, such as buttons, menubuttons, or checkboxes.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Toolbar", component: InnerToolbar }
    ]}
/>

## Accessibility

This component follow WAI-ARIA [design pattern for toolbar](https://www.w3.org/TR/wai-aria-practices-1.1/#toolbar).

## Usage

### Default

A default toolbar.

<Preview>
    <Story name="default">
        <Toolbar align="center">
            <TextInput placeholder="Destination" />
            <RadioGroup>
                <Radio value="roscosmos">Roscosmos</Radio>
                <Radio value="nasa">NASA</Radio>
            </RadioGroup>
            <CheckboxGroup>
                <ToggleButton variant="outline" value="round-trip">Round trip</ToggleButton>
                <ToggleButton variant="outline" value="one-way">One way</ToggleButton>
            </CheckboxGroup>
        </Toolbar>
    </Story>
</Preview>

### Disabled

A toolbar control can be disabled.

<Preview>
    <Story name="disabled control">

            </RadioGroup>
