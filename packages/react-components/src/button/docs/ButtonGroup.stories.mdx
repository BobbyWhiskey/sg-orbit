# ButtonGroup

Related buttons whose actions are related to each other can be group together. Usefull to handle actions in components like [form](?path=/docs/components-form).

<Header
    usage="import { Button } from @orbit-ui/react-components"
    githubPath="/packages/react-components/src/button/src"

### Orientation

A button group can render his buttons vertically.

<Canvas>
    <Story name="vertical">
        <InnerButtonGroup orientation="vertical">
            <Button variant="outline" color="secondary">No, thanks</Button>
            <Button variant="outline" color="secondary">Remind me later</Button>
            <Button color="primary">Rate Now</Button>
        </InnerButtonGroup>
    </Story>
</Canvas>

### Alignment

A button group can change the alignment of his buttons.

<Canvas>
    <Story name="alignment">
        <InnerButtonGroup align="end">
            <Button variant="outline" color="secondary">No, thanks</Button>

        </InnerButtonGroup>
    </Story>
</Canvas>

### Toggle buttons

A button group can contain any types of buttons.
