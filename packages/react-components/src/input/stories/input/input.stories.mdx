## API

A few API functions are available for an input component. To use an API, first get a [ref](https://reactjs.org/docs/refs-and-the-dom.html) on the input component, then call the API function.

### focus

An input can be focused programmatically.

```jsx dark
function MyComponent() {
    const ref = useRef();

    return (
        <div onFocus={() => { ref.focus(); }}>
            <Input ref={ref} />
        </div>
    );
}
```

### select

An input text can be [selected](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement/select) programmatically.

```jsx dark
function MyComponent() {
    const ref = useRef();

    return (
        <Input
            ref={ref}
            onFocus={() => { ref.select(); }}
        />
    );
}
```
