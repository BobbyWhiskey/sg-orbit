# Lozenge

Lozenges are used to highlight an item's status for quick recognition or to give additional fast short information such as highlighting a new item.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Lozenge", component: InnerLozenge }
    ]}
/>

## Slots

Support `"text"` and `"icon"` [slots](?path=/docs/getting-started-slots--page).

## Usage

### Default

A default lozenge.

<Preview>
    <Story name="default">
        <Lozenge>New</Lozenge>
    </Story>
</Preview>

### Icon

A lozenge can contain an [icon](?path=/docs/materials-icons--page).

<Preview>
    <Story name="icon">
        <Lozenge>
            <CheckCircleIcon />
            <Text>New</Text>
        </Lozenge>
        <Lozenge>
            <IconList>
                <CheckCircleIcon /><CheckCircleIcon />
            </IconList>
            <Text>New</Text>
        </Lozenge>
    </Story>
</Preview>

### Size

A lozenge can vary in size.

<Preview>
    <Story name="size">
        <Inline verticalAlign="end">
            <Lozenge size="sm">New</Lozenge>
            <Lozenge>New</Lozenge>
        </Inline>
    </Story>
</Preview>

### Color

A lozenge can use a different color accent.

<Preview>
    <Story name="color">
        <Lozenge color="primary">New</Lozenge>
    </Story>
</Preview>
