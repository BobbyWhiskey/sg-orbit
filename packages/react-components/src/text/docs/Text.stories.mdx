# Text

<Tagline>A primitive component that can be used to render text matching Orbit typography type scale.</Tagline>

<ComponentInfo
    usage={"import { Text } from \"@orbit-ui/react-components\""}
    githubPath="/packages/react-components/src/text/src"
/>

> Only use this component if all other Orbit typographic components don't meet your use case.

## Usage

### Size

You can alter the size of the text by specifying a `size` prop. The available size match Orbit UI typography [type scale](?path=/docs/typography--page#type-scale).

<Canvas>
    <Story name="size">
        <Stack>
            <Text>If two pieces of the same type of metal touch in space they will permanently bond.</Text>
            <Text size="sm">If two pieces of the same type of metal touch in space they will permanently bond.</Text>
        </Stack>
    </Story>
</Canvas>
