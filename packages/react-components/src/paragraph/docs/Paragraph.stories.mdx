## Usage

### Changing font size

You can alter the size of the paragraph by specifying a `size` prop. The available size match Orbit UI [typographic body text](?path=/docs/typography--page#body).

<Canvas>
    <Story name="size">
        <>
            <Paragraph>If two pieces of the same type of metal touch in space they will permanently bond.</Paragraph>
            <Paragraph size="sm">If two pieces of the same type of metal touch in space they will permanently bond.</Paragraph>
        </>
    </Story>
</Canvas>
