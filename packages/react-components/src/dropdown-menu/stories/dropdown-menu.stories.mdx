# Dropdown Menu

A dropdown menu can be used to offer a list of choices.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Dropdown", component: PureDropdownMenu },
        { displayName: "Dropdown.Item", component: PureDropdownMenu.Item },
        { displayName: "Dropdown.Header", component: PureDropdownMenu.Header }
    ]}
/>

## Usage

### Default

A default dropdown menu.

<Preview>
    <Story name="default">
        <DropdownMenu text="File">

</Preview>
