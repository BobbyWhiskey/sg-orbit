# Heading

Heading is used to create various levels of typographic hierarchies.

<dl>
    <div>
        <dt>usage</dt>
        <dd>import Heading from @orbit-ui/react-components</dd>
    </div>
</dl>

## Element type

When using an header, you should specify the element type to render it as. We do recommended using native header elements `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`.

<Preview>
    <Story name="element type">
        <Heading as="h1">Migrate, adapt, and control the cloud.</Heading>
    </Story>
</Preview>

## Changing font size

You can alter the font size of the heading by specifying a `size` prop. The available size match Orbit UI [typographic headlines](?path=/docs/materials-typography--page#headlines).

<Preview>
    <Story name="headers">
        <div>
            <Heading as="h3">Migrate, adapt, and control the cloud.</Heading>
            <Heading as="h4" size="sm">Migrate, adapt, and control the cloud.</Heading>
        </div>
    </Story>
</Preview>
