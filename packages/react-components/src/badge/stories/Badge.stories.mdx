# Badge

A badge is a floating component displaying a notification such as a count.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Badge", component: InnerBadge }
    ]}
/>

## Usage

### Default

A default badge.

<Preview>
    <Story name="default">
        <Inline gap={8} align="center">
            <Badge>
                <Text>100</Text>
                <Square />
            </Badge>
            <Badge overlap="circle">
                <Text>100</Text>
                <Circle />
            </Badge>
        </Inline>
    </Story>
</Preview>

### Size

A badge can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <Inline gap={8} align="end">
            <Badge size="small" overlap="circle">
                <Text>100</Text>
                <Circle />
            </Badge>
            <Badge overlap="circle">
                <Text>100</Text>
                <Circle />
            </Badge>
            <Badge size="large" overlap="circle">
                <Text>100</Text>
                <Circle />
            </Badge>
        </Inline>
    </Story>
</Preview>

### Dot

A badge can be rendered as a dot. A single digit is supported.

<Preview>
    <Story name="dot">
        <Inline gap={8} align="center">
            <Badge variant="dot" overlap="circle">
                <Text>5</Text>
                <Circle />
            </Badge>
            <Badge variant="dot" overlap="icon">
                <EmailIcon size="large" />
            </Badge>
            <Badge variant="dot" >
                Notifications
            </Badge>
        </Inline>
    </Story>
</Preview>

### Icon

A badge can be an icon.

<Preview>
    <Story name="icon">
        <Badge variant="icon" overlap="circle">
            <CheckCircleIcon />
            <Circle />
        </Badge>
    </Story>
</Preview>
