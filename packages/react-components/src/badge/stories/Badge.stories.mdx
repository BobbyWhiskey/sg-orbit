# Badge

A badge is a floating component displaying a notification such as a count.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Badge", component: InnerBadge }
    ]}
/>

## Usage

### Default

A default badge.

<Preview>
    <Story name="default">
        <Inline gap={8} align="center">
            <Badge content="100">
                <Square />
            </Badge>
            <Badge content="100" overlap="circle">
                <Circle />
            </Badge>
        </Inline>
    </Story>
</Preview>

### Size

A badge can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <Inline gap={8} align="end">
            <Badge size="small" content="100" overlap="circle">
                <Circle />
            </Badge>
            <Badge content="100" overlap="circle">
                <Circle />
            </Badge>
            <Badge size="large" content="100" overlap="circle">
                <Circle />
            </Badge>
        </Inline>
    </Story>
</Preview>

### Dot

A badge can be rendered as a dot. A single digit is supported.

<Preview>
    <Story name="dot">
        <Inline gap={8} align="center">
            <Inline>
                <Badge variant="dot" size="small" overlap="circle">
                    <Circle />
                </Badge>
                <Badge variant="dot" overlap="circle">
                    <Circle />
                </Badge>
                <Badge variant="dot" size="large" overlap="circle">
                    <Circle />
                </Badge>
            </Inline>
            <Inline>
                <Badge variant="dot" size="small" content="5">
                    <EmailIcon size="large" />
                </Badge>
                <Badge variant="dot" content="5">
                    <EmailIcon size="large" />
                </Badge>
                <Badge variant="dot" size="large" content="5">
                    <EmailIcon size="large" />
                </Badge>
            </Inline>
        </Inline>
    </Story>
</Preview>

### Icon

A badge can be an icon.

<Preview>
    <Story name="icon">
        <Inline align="end">
            <Badge variant="icon" size="small" content={<CheckCircleIcon />} overlap="circle">
                <Circle />
            </Badge>
            <Badge variant="icon" content={<CheckCircleIcon />} overlap="circle">
                <Circle />
            </Badge>
            <Badge variant="icon" size="large" content={<CheckCircleIcon />} overlap="circle">
                <Circle />
            </Badge>
        </Inline>
    </Story>
</Preview>
