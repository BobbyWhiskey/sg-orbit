# Badge

A badge is a floating component displaying a notification such as a count.

<ComponentInfo
    usage="import { Badge } from @orbit-ui/react-components"
    githubPath="/packages/react-components/src/badge/src"
/>

## Usage

### Default

A default badge.

<Canvas>
    <Story name="default">
        <Inline gap={8} verticalAlign="center">
            <Badge>
                <Text>5</Text>
                <Square />
            </Badge>
            <Badge>
                <Text>5+</Text>
                <Square />
            </Badge>
            <Badge>
                <Text>50</Text>
                <Square />
            </Badge>
            <Badge>
                <Text>500</Text>
                <Square />
            </Badge>
        </Inline>
    </Story>
</Canvas>

### Dot

A badge can be rendered as a dot. A single digit is supported.

<Canvas>
    <Story name="dot">
        <Inline gap={8} verticalAlign="center">
            <Badge variant="dot" overlap="circle">
                <Circle />
            </Badge>
            <Badge variant="dot" overlap="icon">
                <EmailIcon size="lg" />
            </Badge>
            <Badge variant="dot">
                Notifications
            </Badge>
        </Inline>
    </Story>
</Canvas>

### Icon

A badge can be an [icon](?path=/docs/icon-gallery--page).

<Canvas>
    <Story name="icon">
        <Badge variant="icon" overlap="circle">
            <CheckCircleIcon />
            <Circle />
        </Badge>
    </Story>
</Canvas>

### Overlap

A badge can overlap over elements with different shape.

<Canvas>
    <Story name="overlap">
        <Inline gap={13} verticalAlign="center">
            <Inline gap={8}>
                <Badge>
                    <Text>100</Text>
                    <Square />
                </Badge>
                <Badge variant="dot">
                    <Square />
                </Badge>
                <Badge variant="icon">
                    <CheckCircleIcon />
                    <Square />
                </Badge>
            </Inline>
            <Inline gap={8}>
                <Badge overlap="circle">
                    <Text>100</Text>
                    <Circle />
                </Badge>
                <Badge variant="dot" overlap="circle">
                    <Circle />
                </Badge>
                <Badge variant="icon" overlap="circle">
                    <CheckCircleIcon />
                    <Circle />
                </Badge>
            </Inline>
            <Inline gap={8}>
                <Badge overlap="icon">
                    <Text>100</Text>
                    <EmailIcon size="lg" />
                </Badge>
                <Badge variant="dot" overlap="icon">
                    <EmailIcon size="lg" />
                </Badge>
                <Badge variant="icon" overlap="icon">
                    <CheckCircleIcon />
                    <EmailIcon size="lg" />
                </Badge>
            </Inline>
        </Inline>
    </Story>
</Canvas>
