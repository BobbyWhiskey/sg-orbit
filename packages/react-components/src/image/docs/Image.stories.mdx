## Accessibility

An image must have an `alt` attribute.

## Usage

### Shape

An image can use a different shape.

<Canvas>
    <Story name="shape">
        <Inline>
            <Image shape="rounded" src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            <Image shape="circular" src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
        </Inline>
    </Story>
</Canvas>

### Size

An image can adjust it's size by using the `size` prop or the native `width` and `height` props.

<Canvas>
    <Story name="size">
        <Image size="200px" src={LaunchImage} alt="SpaceX launch" />
    </Story>
</Canvas>
