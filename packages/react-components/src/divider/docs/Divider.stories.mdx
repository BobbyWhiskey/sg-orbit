# Divider

A divider to separates and distinguishes sections of content or groups of menuitems.

<dl>
    <div>
        <dt>usage</dt>
        <dd>import Divider from @orbit-ui/react-components</dd>
    </div>
</dl>

## Usage

### Horizontal

By default a divider render horizontally and takes the width of it's container.

<Preview>
    <Story name="horizontal">
        <Divider />
    </Story>
</Preview>

### Vertical

A divider can change his orientation to vertical.

<Preview>
    <Story name="vertical">
        <Box className="flex">
            <Dot color="success-500">Starlink operational</Dot>
            <Divider orientation="vertical" />
            <TextLink href="#">Manage</TextLink>
            <Divider orientation="vertical" />
            <TextLink href="#">View logs</TextLink>
        </Box>
    </Story>
</Preview>

### Labelled

A divider can have a label.

<Preview>
    <Story name="labelled">
        <Divider>Last 30 days</Divider>
    </Story>
</Preview>
