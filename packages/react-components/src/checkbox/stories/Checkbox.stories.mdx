# Checkbox

A checkbox allows a user to select a value from a small set of options, often binary.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Checkbox", component: InnerCheckbox }
    ]}
/>

## Usage

### Label

A checkbox can have a label.

<Preview>
    <Story name="label">
        <Checkbox label="Milky Way" />
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked label="Milky Way" />
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate label="Milky Way" />
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled label="Milky Way" />
    </Story>
</Preview>

### Readonly

A checkbox can be readonly.

<Preview>
    <Story name="readonly">
        <Checkbox readOnly label="Milky Way" />
    </Story>
</Preview>

### No label

A checkbox can be rendered without a label.

<Preview>
    <Story name="no label">
        <Checkbox />
    </Story>
</Preview>

### Icons

A checkbox can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Checkbox label="Milky Way" icons={<InfoIcon />} />
            <Checkbox label="Andromeda" icons={[<InfoIcon />, <HelpIcon />]} />
        </div>
    </Story>
</Preview>

### Badge

A checkbox can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <div className="flex items-lot-of-space-between">
            <Checkbox label="Milky Way" badge={<Badge>6</Badge>} />
            <Checkbox label="Milky Way" badge={<Badge variant="inline">6</Badge>} />
        </div>
    </Story>
</Preview>

### Size

A checkbox can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Checkbox size="small" label="Milky Way" />
            <Checkbox label="Milky Way" />
            <Checkbox size="large" label="Milky Way" />
        </div>
    </Story>
</Preview>
