# Checkbox

A checkbox allows a user to select a value from a small set of options, often binary.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Checkbox", component: InnerCheckbox }
    ]}
/>

## Usage

<Preview>
    <Story name="default">
        <Checkbox>Milky Way</Checkbox>
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked>Milky Way</Checkbox>
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate>Milky Way</Checkbox>
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled>Milky Way</Checkbox>
    </Story>
</Preview>

### Readonly

A checkbox can be readonly.

<Preview>
    <Story name="readonly">
        <Checkbox readOnly>Milky Way</Checkbox>
    </Story>
</Preview>

### No label

A checkbox can be rendered without a label.

<Preview>
    <Story name="no label">
        <Checkbox />
    </Story>
</Preview>

### Icon

A checkbox can contain [icons](?path=/docs/materials-icons--page) after his label.

<Preview>
    <Story name="icons">
        <Stack>
            <Checkbox icon={<InfoIcon />}>Milky Way</Checkbox>
            <Checkbox icon={<IconGroup><InfoIcon /><HelpIcon /></IconGroup>}>Andromeda</Checkbox>
        </Stack>
    </Story>
</Preview>

### Badge

A checkbox can have a [badge](?path=/docs/components-badge--default-story) after his label.

<Preview>
    <Story name="badge">
        <Stack>
            <Checkbox badge={<Badge>6</Badge>}>Milky Way</Checkbox>
            <Checkbox badge={<Badge variant="inline">6</Badge>}>Andromeda</Checkbox>
        </Stack>
    </Story>
</Preview>

### Size

A checkbox can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Checkbox size="small">Milky Way</Checkbox>
            <Checkbox>Milky Way</Checkbox>
            <Checkbox size="large">Milky Way</Checkbox>
        </div>
    </Story>
</Preview>

### Reverse

A checkbox checkmark can be to the left or to the right of his label.

<Preview>
    <Story name="reverse">
        <Checkbox reverse icon={<InfoIcon />} badge={<Badge variant="inline">6</Badge>}>Milky Way</Checkbox>
    </Story>
</Preview>
