# Checkbox

A checkbox allows the user to toggle between 2 or 3 (indeterminate) choices.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Checkbox", component: InnerCheckbox },
        { displayName: "CheckboxGroup", component: InnerCheckboxGroup }
    ]}
/>

## Accessibility

Follow [WAI-ARIA design pattern for checkbox](https://www.w3.org/TR/wai-aria-practices/#checkbox).

## Slots

Support `"label"`, `"icon"` and `"counter"` [slots](?path=/docs/getting-started-slots--page).

## Usage

### Label

A default checkbox.

<Preview>
    <Story name="default">
        <Checkbox>Milky Way</Checkbox>
    </Story>
</Preview>

### No label

A checkbox can be rendered without a label.

<Preview>
    <Story name="no label">
        <Checkbox />
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked>Milky Way</Checkbox>
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate>Milky Way</Checkbox>
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled>Milky Way</Checkbox>
    </Story>
</Preview>

### Icon

A checkbox can have [icons](?path=/docs/materials-icons--page) after his label.

<Preview>
    <Story name="icon">
        <Checkbox>
            <Text>Milky Way</Text>
            <InfoIcon />
        </Checkbox>
    </Story>
</Preview>

### Counter

A checkbox can have a [counter](?path=/docs/components-counter--default-story) after his label.

<Preview>
    <Story name="counter">
        <Inline>
            <Checkbox>
                <Text>Milky Way</Text>
                <Counter>6</Counter>
            </Checkbox>
            <Checkbox>
                <Text>Andromeda</Text>
                <Counter variant="divider">6</Counter>
            </Checkbox>
        </Inline>
    </Story>
</Preview>

### Size

A checkbox can vary in size.

<Preview>
    <Story name="size">
        <Inline verticalAlign="end">
            <Checkbox size="sm">Milky Way</Checkbox>
            <Checkbox>Milky Way</Checkbox>
        </Inline>
    </Story>
</Preview>

### Reverse

A checkbox checkmark can be to the left or to the right of his label.

<Preview>
    <Story name="reverse">
        <Checkbox reverse>
            <Text>Milky Way</Text>
            <InfoIcon />
            <Counter variant="divider">6</Counter>
        </Checkbox>
    </Story>
</Preview>

### Validation

A checkbox can display a validation state to communicate to the user whether the current value is valid or invalid. Implement your own validation logic in your app and pass either `"valid"` or `"invalid"` to the input via the `validationState` prop.

The example below illustrates how one would validate if the user has checked the checkbox.

<FilePreview filePath="/checkbox/stories/CheckedValidation">
    <CheckedValidation />
</FilePreview>

### Render Props

A checkbox can update is content based on the `isChecked` value by providing a render prop function.

<Preview>
    <Story name="render props">
        <Checkbox>
            {
                ({ isChecked }) => {
                    return isChecked ? "On" : "Off";
                }
            }
        </Checkbox>
    </Story>
</Preview>

## Checkbox Group

Related checkboxes can be group together.

<Preview>
    <Story name="checkbox group">
        <CheckboxGroup>

            <Checkbox value="medusa">Medusa</Checkbox>
        </CheckboxGroup>
    </Story>

### Toggle Buttons

In somes cases, you might want a group of toggle buttons to behave like a checkbox group.

### Render Props

A checkbox groups can render his items with a render prop function.

<Preview>
    <Story name="group render props">
        <CheckboxGroup>
            {
                ({ checkedValue }) =>
                    ["Milky Way", "Andromeda", "Medusa"].map(x =>
                        <Checkbox
                            className={checkedValue && checkedValue.includes(x) ? "primary-500" : undefined}
                            value={x}
                        >
                            {x}
                        </Checkbox>
                    )
            }
        </CheckboxGroup>
    </Story>
</Preview>
