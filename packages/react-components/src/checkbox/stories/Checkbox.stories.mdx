# Checkbox

A checkbox allows the user to toggle between 2 or 3 (indeterminate) choices.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Checkbox", component: InnerCheckbox },
        { displayName: "CheckboxGroup", component: InnerCheckboxGroup }
    ]}
/>

## Accessibility

This component follow WAI-ARIA [design pattern for checkbox](https://www.w3.org/TR/wai-aria-practices-1.1/#checkbox).

## Usage

<Preview>
    <Story name="default">
        <Checkbox>Milky Way</Checkbox>
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked>Milky Way</Checkbox>
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate>Milky Way</Checkbox>
    </Story>
</Preview>

### No label

A checkbox can be rendered without a label.

<Preview>
    <Story name="no label">
        <Checkbox />
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled>Milky Way</Checkbox>
    </Story>
</Preview>

### Readonly

A checkbox can be readonly.

<Preview>
    <Story name="readonly">
        <Checkbox readOnly>Milky Way</Checkbox>
    </Story>
</Preview>

### Icon

A checkbox can have [icons](?path=/docs/materials-icons--page) after his label.

<Preview>
    <Story name="icon">
        <Inline>
            <Checkbox icon={<InfoIcon />}>Milky Way</Checkbox>
            <Checkbox icon={<IconGroup><InfoIcon /><HelpIcon /></IconGroup>}>Andromeda</Checkbox>
        </Inline>
    </Story>
</Preview>

### Badge

A checkbox can have a [badge](?path=/docs/components-badge--default-story) after his label.

<Preview>
    <Story name="badge">
        <Inline>
            <Checkbox badge={<Badge>6</Badge>}>Milky Way</Checkbox>
            <Checkbox badge={<Badge variant="inline">6</Badge>}>Andromeda</Checkbox>
        </Inline>
    </Story>
</Preview>

### Size

A checkbox can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <Inline align="end">
            <Checkbox size="small">Milky Way</Checkbox>
            <Checkbox>Milky Way</Checkbox>
            <Checkbox size="large">Milky Way</Checkbox>
        </Inline>
    </Story>
</Preview>

### Reverse

A checkbox checkmark can be to the left or to the right of his label.

<Preview>
    <Story name="reverse">
        <Checkbox reverse icon={<InfoIcon />} badge={<Badge variant="inline">6</Badge>}>Milky Way</Checkbox>
    </Story>
</Preview>

### Render Props

A checkbox can update is content with a render props based on the state values.

<Preview>
    <Story name="render props">
        <Checkbox>
            {
                ({ isChecked }) => {
                    return isChecked ? "On" : "Off";
                }
            }
        </Checkbox>
    </Story>
</Preview>

## Checkbox Group

Related checkboxes can be group together.

<Preview>
    <Story name="checkbox group">
        <CheckboxGroup>
            <Checkbox value="milky-way">Milky Way</Checkbox>
            <Checkbox value="andromeda">Andromeda</Checkbox>
            <Checkbox value="medusa">Medusa</Checkbox>
        </CheckboxGroup>
    </Story>
</Preview>

### Label

A checkbox group can have a label.

<Preview>
    <Story name="labelled checkbox group">
        <CheckboxGroup label="Your favorite galaxies">
            <Checkbox value="milky-way">Milky Way</Checkbox>
            <Checkbox value="andromeda">Andromeda</Checkbox>
            <Checkbox value="medusa">Medusa</Checkbox>
        </CheckboxGroup>
    </Story>
</Preview>

### Orientation

A checkbox group can change his orientation to `vertical` to render his elements vertically.

<Preview>
    <Story name="orientation">
        <CheckboxGroup orientation="vertical">
            <Checkbox value="milky-way" >Milky Way</Checkbox>
            <Checkbox value="andromeda">Andromeda</Checkbox>
            <Checkbox value="medusa">Medusa</Checkbox>
        </CheckboxGroup>
    </Story>
</Preview>

### Toggle Buttons

In somes cases, you might want a group of toggle buttons to behave like a checkbox group.

<Preview>
    <Story name="toggle buttons">
        <CheckboxGroup gap={2}>
            <ToggleButton value="none" variant="outline">None</ToggleButton>
            <ToggleButton value="1" variant="outline" shape="circular">1</ToggleButton>
            <ToggleButton value="2" variant="outline" shape="circular">2</ToggleButton>
            <ToggleButton value="3" variant="outline" shape="circular">3</ToggleButton>
            <ToggleButton value="4+" variant="outline" shape="circular">4+</ToggleButton>
        </CheckboxGroup>
    </Story>
</Preview>

### Render Props

A checkbox groups can render his items with a render props.

<Preview>
    <Story name="group render props">
        <CheckboxGroup>
            {
                ({ checkedValue }) =>
                    ["Milky Way", "Andromeda", "Medusa"].map(x =>
                        <Checkbox
                            className={checkedValue && checkedValue.includes(x) ? "primary-500" : undefined}
                            value={x}
                        >
                            {x}
                        </Checkbox>
                    )
            }
        </CheckboxGroup>
    </Story>
</Preview>
