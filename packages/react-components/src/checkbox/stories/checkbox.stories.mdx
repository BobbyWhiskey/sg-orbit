# Checkbox

A checkbox allows a user to select a value from a small set of options, often binary.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Checkbox", component: InnerCheckbox }
    ]}
/>

## Usage

### Text

A checkbox can have text.

<Preview>
    <Story name="text">
        <Checkbox text="Milky Way" />
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked text="Milky Way" />
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate text="Milky Way" />
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled text="Milky Way" />
    </Story>
</Preview>

### Readonly

A checkbox can be readonly.

<Preview>
    <Story name="readonly">
        <Checkbox readOnly text="Milky Way" />
    </Story>
</Preview>

### No text

A checkbox doesn't require to have text.

<Preview>
    <Story name="no text">
        <Checkbox />
    </Story>
</Preview>

### Icons

A checkbox can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text. Specify the icons to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Checkbox text="Milky Way" icons={<InfoIcon />} />
            <Checkbox text="Andromeda" icons={[<InfoIcon />, <HelpIcon />]} />
        </div>
    </Story>
</Preview>

### Label

A checkbox can have a [label](?path=/docs/components-label--default-story) after his text. Specify the label to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="label">
        <Checkbox text="Milky Way" label={<Label>6</Label>} />
    </Story>
</Preview>

### Count

A checkbox can have a [count](?path=/story/components-count--default-story) after his text. Specify the count to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="count">
        <Checkbox text="Milky Way" count={<Count>6</Count>} />
    </Story>
</Preview>

### Size

A checkbox can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Checkbox size="small" text="Milky Way" />
            <Checkbox text="Milky Way" />
            <Checkbox size="large" text="Milky Way" />
        </div>
    </Story>
</Preview>
