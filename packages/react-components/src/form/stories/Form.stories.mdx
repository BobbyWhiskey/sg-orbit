# Form

Forms are commonly used to provide user interaction in web applications.

## Installation

<InstallReactComponents imports={<Imports />}/>

## Props

<Props
    componentsDefinitions={[
        { displayName: "Form", component: InnerForm }
    ]}
/>

## Accessibility

When combined with [field](?path=/story/components-field--input) components, a form component follows the [WAI specifications for forms](https://www.w3.org/WAI/tutorials/forms/).

## Usage

In additional to wrapping a form element, a form component provide layout for fields and buttons. **For layout to be applied correctly, every inputs must be wrapped in a field component.**

### Vertical layout

A form can stack fields vertically.

<Preview>
    <Story name="vertical layout">
        <Form fluid className="w-25">
            <Field required>
                <Label>Username</Label>
                <TextInput placeholder="john@spacex.com" />
            </Field>
            <Field required>
                <Label>Password</Label>
                <PasswordInput />
                <HelpMessage>I bet it's a strong one!</HelpMessage>
            </Field>
            <Field>
                <Checkbox>Remember me</Checkbox>
            </Field>
            <Button type="submit">
                Login
            </Button>
        </Form>
    </Story>
</Preview>
