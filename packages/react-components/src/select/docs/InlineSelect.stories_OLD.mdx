# Inline Select

An inlined select can appear inline with other content.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Select", component: InnerSelect },
        { displayName: "Select.Item", component: InnerSelect.Item }
    ]}
/>

## Usage

<WarningMessage>
    This component <i>mostly</i> support the same feature set as the <a href="?path=/docs/components-select-standard--default-story">standard select</a> component.
</WarningMessage>

### Constants

The following constants are used in the examples below:

export const ASTRONAUTS = [
    selectItem("David Saint-Jacques", "David Saint-Jacques"),
    selectItem("Chris Hadfield", "Chris Hadfield"),
    selectItem("Jeremy Hansen", "Jeremy Hansen"),
    selectItem("Jenni Sidey", "Jenni Sidey"),
    selectItem("Robert Thirks", "Robert Thirks"),
    selectItem("Julie Payette", "Julie Payette")
];

```javascript dark
const ASTRONAUTS = [
    selectItem("David Saint-Jacques", "David Saint-Jacques"),
    selectItem("Chris Hadfield", "Chris Hadfield"),
    selectItem("Jeremy Hansen", "Jeremy Hansen"),
    selectItem("Jenni Sidey", "Jenni Sidey"),
```

### Default

A default inlined select.

<Preview>
    <Story name="default">
        <Select
            placeholder="select an astronaut"
            inline
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Selected value

A inline select can have an initial value

<Preview>
    <Story name="selected value">
        <Select
            defaultValue="Chris Hadfield"
            inline
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### In a block

An inlined select can be embedded in a block.

<Preview>
    <Story name="in a block">
        <div>Canadian astronaut <Select
            placeholder="select an astronaut"
            inline
            options={ASTRONAUTS}
        />, the first Canadian Commander of the ISS, would go on to achieve worldwide fame in 2013 for releasing a music video he recorded on the International Space Station of his version of David Bowie's song "Space Oddity".</div>
    </Story>
</Preview>

### Icon

An inline select can display an [icon](?path=/docs/materials-icons--page) before it's content.

<Preview>
    <Story name="icon">
        <Select
            placeholder="select an astronaut"
            icon={<GroupIcon />}
            inline
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Disabled

An inlined select can be disabled.

<Preview>
    <Story name="disabled">
        <Select
            placeholder="select an astronaut"
            inline
            disabled
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Loading

An inlined select can show that it is currently loading data.
