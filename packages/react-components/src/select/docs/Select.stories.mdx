# Select

<Tagline>A select allow users to choose a single option from a collapsible list of options.</Tagline>

<ComponentInfo
    usage={"import { Select, Item, Section } from \"@orbit-ui/react-components\""}
    githubPath="/packages/react-components/src/select/src"
/>

A select is a combination of a [Button](?path=/docs/button--default-story), a [Popover](?path=/docs/popover--default-story) and a [Listbox](?path=/docs/listbox--default-story) to form a collapsible dropdown listbox. You can find a detailed example on [WAI-ARIA](https://www.w3.org/TR/wai-aria-practices-1.1/examples/listbox/listbox-collapsible.html).

## Usage

### Default

A default select.

<Preview>
    <Story name="default">
        <Select placeholder="Planets" aria-label="Planets">
            <Item key="earth">Earth</Item>
            <Item key="jupiter">Jupiter</Item>
            <Item key="mars">Mars</Item>
            <Item key="mercury">Mercury</Item>
            <Item key="neptune">Neptune</Item>
            <Item key="saturn">Saturn</Item>
            <Item key="uranus">Uranus</Item>
        </Select>
    </Story>
</Preview>

### Section

A select items can be group by sections.

<Preview>
    <Story name="section">
        <Select placeholder="Planets" aria-label="Planets">
            <Section title="Visited">
                <Item key="earth">Earth</Item>
                <Item key="mars">Mars</Item>
                <Item key="saturn">Saturn</Item>
            </Section>
            <Section title="Not Visited">
                <Item key="jupiter">Jupiter</Item>
                <Item key="mercury">Mercury</Item>
                <Item key="neptune">Neptune</Item>
                <Item key="uranus">Uranus</Item>
            </Section>
        </Select>
    </Story>
</Preview>

### Icon

A select item can contain [icons](?path=/docs/icon-gallery--page).

<Preview>
    <Story name="icon">
        <Select placeholder="Planets" aria-label="Planets">

                <NotificationIcon />

            </Item>

                <HelpIcon />

            <Item key="mars">
                <LightbulbIcon />
                <Text>Mars</Text>

        </Select>
    </Story>
</Preview>

### Right icon

A select item can contain *non standard* right [icons](?path=/docs/icon-gallery--page) can be provided to handle special cases like displaying a list of icons. However, think twice before adding right icons, left icons should be your go to.

<Preview>
    <Story name="right icon">
        <Select placeholder="Planets" aria-label="Planets">
            <Item key="earth">
                <Text>Earth</Text>
                <IconList slot="right-icon">
                    <NotificationIcon />
                    <HelpIcon />
                    <LightbulbIcon />
                </IconList>
            </Item>
            <Item key="jupiter">
                <Text>Jupiter</Text>
                <HelpIcon slot="right-icon" />
            </Item>
            <Item key="mars">
                <Text>Mars</Text>
                <IconList slot="right-icon">
                    <NotificationIcon />
                    <HelpIcon />
                    <LightbulbIcon />
                </IconList>
            </Item>
        </Select>
    </Story>
</Preview>

### Validation

A select can display a validation state to communicate to the user whether the current value is valid or invalid. Implement your own validation logic in your app and pass either `"valid"` or `"invalid"` to the input via the `validationState` prop.

The example below illustrates how one would validate that a specific value have been selected.
