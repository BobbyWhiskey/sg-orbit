# Select Item

An item of a select component.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Select.Item", component: InnerSelect.Item }
    ]}
/>

## Usage

### Constants

The following constants are used in the examples below:

export const DAVID_SAINT_JACQUES_ITEM = selectItem("David Saint-Jacques", "David Saint-Jacques");
export const CHRIS_HADFIELD_ITEM = selectItem("Chris Hadfield", "Chris Hadfield");
export const JEREMY_HANSEN_ITEM = selectItem("Jeremy Hansen", "Jeremy Hansen");
export const JENNI_SIDEY_ITEM = selectItem("Jenni Sidey", "Jenni Sidey");
export const ROBERT_THIRKS_ITEM = selectItem("Robert Thirks", "Robert Thirks");
export const JULIE_PAYETTE_ITEM = selectItem("Julie Payette", "Julie Payette");

export const ASTRONAUTS = [
    DAVID_SAINT_JACQUES_ITEM,
    CHRIS_HADFIELD_ITEM,
    JEREMY_HANSEN_ITEM,
    JENNI_SIDEY_ITEM,
    ROBERT_THIRKS_ITEM,
    JULIE_PAYETTE_ITEM
];

### Text

A select item can have a text content.

<Preview>
    <Story name="text">
        <Select
            placeholder="Astronaut"
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Description

A select item can have a description.

<Preview>
    <Story name="description">
        <Select
            placeholder="Astronaut"
            options={[
                DAVID_SAINT_JACQUES_ITEM,
                selectItem("Chris Hadfield", "Chris Hadfield", { description: "Commander of the ISS" }),

                JENNI_SIDEY_ITEM,
                ROBERT_THIRKS_ITEM,
                selectItem("Julie Payette", "Julie Payette", { description: "29th GGC" })
            ]}
        />
    </Story>
</Preview>
