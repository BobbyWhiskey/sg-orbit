# Multiple Values Select

A select can allow a user to make multiple choices.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Select", component: InnerSelect },
        { displayName: "Select.Item", component: InnerSelect.Item }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-select-standard--default-story">standard select</a> component.
</WarningMessage>

### Default

A default multiple values select.

<Preview>
    <Story name="default">
        <Select
            multiple
            options={GALAXIES}
        />
    </Story>
</Preview>

### Selected values

A multiple values select can have initial values. _Dont forget to specify values has an array_.

<Preview>
    <Story name="selected values">
        <Select
            multiple
            defaultValue={["Comet"]}
            options={GALAXIES}
        />
    </Story>
</Preview>

### Icon

A multiple values select can display an [icon](?path=/docs/materials-icons--page) before it's content.

<Preview>
    <Story name="icon">
        <Select
            placeholder="Select a galaxy"
            multiple
            icon={<MagnifierIcon />}
            options={GALAXIES}
        />
    </Story>
</Preview>

### Disabled

A multiple values select can be disabled.

<Preview>
