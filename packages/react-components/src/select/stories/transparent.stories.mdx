# Transparent Select

A select can be transparent in order to be more discrete.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Select", component: PureSelect },
        { displayName: "Select.Item", component: PureSelect.Item }
    ]}
/>

### Default

A default transparent select.

<Preview>
    <Story name="default">
        <Select
            transparent
            placeholder="Select a galaxy"
            options={GALAXIES}
        />
    </Story>
</Preview>

### Selected values

A transparent select can have initial values. Dont forget to specify values has an array_.

<Preview>
    <Story name="selected values">
        <Select
            transparent
            placeholder="Select a galaxy"
            defaultValue={["Comet"]}
            options={GALAXIES}
        />
    </Story>
</Preview>

### Icon

A transparent select can display an [icon](?path=/docs/materials-icons--page) before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <Select
            placeholder="Select a galaxy"
            transparent
            icon={<MagnifierIcon />}
            options={GALAXIES}
        />
    </Story>
</Preview>

### Disabled

A transparent select can be disabled.

<Preview>
    <Story name="disabled">
        <Select
            placeholder="Select a galaxy"
            transparent
            disabled
            options={GALAXIES}
        />
    </Story>
</Preview>

### Fluid

A transparent select can take all the space available.

<Preview>
    <Story name="fluid">
        <Select
            placeholder="Select a galaxy"
            transparent
            fluid
            options={GALAXIES}
        />
    </Story>
</Preview>

### Loading

A transparent select can show that it is currently loading data.

<Preview>
    <Story name="loading">
        <Select
            placeholder="Select a galaxy"
            transparent
            loading
            options={GALAXIES}
        />
    </Story>
</Preview>
