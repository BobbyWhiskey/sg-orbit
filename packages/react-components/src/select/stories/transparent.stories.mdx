# Transparent Select

A select can be transparent in order to be more discrete.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Select", component: PureSelect },
        { displayName: "Select.Item", component: PureSelect.Item }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-select-standard--default-story">standard select</a> component.
</WarningMessage>

### Constants

The following constants are used in the examples below:

export const ASTRONAUTS = [
    selectItem("David Saint-Jacques", "David Saint-Jacques"),
    selectItem("Chris Hadfield", "Chris Hadfield"),
    selectItem("Jeremy Hansen", "Jeremy Hansen"),
    selectItem("Jenni Sidey", "Jenni Sidey"),
    selectItem("Robert Thirks", "Robert Thirks"),
    selectItem("Julie Payette", "Julie Payette")
];

```javascript dark
const ASTRONAUTS = [
    selectItem("David Saint-Jacques", "David Saint-Jacques"),
    selectItem("Chris Hadfield", "Chris Hadfield"),
```

### Default

A default transparent select.

<Preview>
    <Story name="default">
        <Select
            transparent
            placeholder="Select a galaxy"
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Selected value

A transparent select can have an initial value.

<Preview>
    <Story name="selected values">
        <Select
            transparent
            placeholder="Select a galaxy"
            defaultValue="John Glen"
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Icon

A transparent select can display an [icon](?path=/docs/materials-icons--page) before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <Select
            placeholder="Select a galaxy"
            transparent
            icon={<MagnifierIcon />}
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Disabled

A transparent select can be disabled.

<Preview>
    <Story name="disabled">
        <Select
            placeholder="Select a galaxy"
            transparent
            disabled
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Clearable

A clearable transparent select let a user remove is selection.

<Preview>
    <Story name="clearable">
        <Select
            placeholder="Astronaut"
            clearable
            transparent
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Fluid

A transparent select can take all the space available.

<Preview>
    <Story name="fluid">
        <Select
            placeholder="Select a galaxy"
            transparent
            fluid
            options={ASTRONAUTS}
        />
    </Story>
</Preview>

### Loading

A transparent select can show that it is currently loading data.

<Preview>
