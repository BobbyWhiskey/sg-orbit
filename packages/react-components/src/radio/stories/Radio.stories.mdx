# Radio

Radios allow a user to select one option from a set.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Radio", component: InnerRadio }
    ]}
/>

## Usage

### Default

A default radio.

<Preview>
    <Story name="default">
        <Radio>Mars</Radio>
    </Story>
</Preview>

### Selected

A radio can be selected.

<Preview>
    <Story name="selected">
        <Radio defaultChecked>Mars</Radio>
    </Story>
</Preview>

### No label

A radio can be rendered without a label.

<Preview>
    <Story name="no label">
        <Radio />
    </Story>
</Preview>

### Disabled

A radio can be disabled.

<Preview>
    <Story name="disabled">
        <Radio disabled>Mars</Radio>
    </Story>
</Preview>

### Readonly

A radio can be readonly.

<Preview>
    <Story name="readonly">
        <Radio readOnly>Mars</Radio>
    </Story>
</Preview>

### Icons

A checkbox can have [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icon">
        <Inline>
            <Radio icon={<LightbulbIcon />}>Mars</Radio>
            <Radio icon={<IconGroup><LightbulbIcon /><LightbulbIcon /></IconGroup>}>Mars</Radio>
        </Inline>
    </Story>
</Preview>

### Badge

A radio can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <Inline>
            <Radio badge={<Badge>6</Badge>}>Mars</Radio>
            <Radio badge={<Badge variant="inline">6</Badge>}>Mars</Radio>
        </Inline>
    </Story>
</Preview>

### Size

A radio can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <Inline align="end">
            <Radio size="small">Mars</Radio>
            <Radio>Mars</Radio>
            <Radio size="large">Mars</Radio>
        </Inline>
    </Story>
</Preview>
