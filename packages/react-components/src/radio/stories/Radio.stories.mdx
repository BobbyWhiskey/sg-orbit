# Radio

Radio buttons allow a user to select one option from a set.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Radio", component: InnerRadio }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-checkbox--text">checkbox</a> component.
</WarningMessage>

### Text

A radio can have a text.

<Preview>
    <Story name="text">
        <Radio text="Mars"/>
    </Story>
</Preview>

### Selected

A radio can be selected.

<Preview>
    <Story name="selected">
        <Radio defaultChecked text="Mars"/>
    </Story>
</Preview>

### Disabled

A radio can be disabled.

<Preview>
    <Story name="disabled">
        <Radio disabled text="Mars"/>
    </Story>
</Preview>

### Readonly

A radio can be readonly.

<Preview>
    <Story name="readonly">
        <Radio readOnly text="Mars" />
    </Story>
</Preview>

### No text

A radio doesn't require a text.

<Preview>
    <Story name="no text">
        <Radio />
    </Story>
</Preview>

### Icons

A radio can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text. Specify the icons to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Radio text="Milky Way" icons={<LightbulbIcon />} />
            <Radio text="Milky Way" icons={[<LightbulbIcon />, <InfoIcon />]} />
        </div>
    </Story>
</Preview>

### Label

A radio can have a [label](?path=/docs/components-label--default-story) after his text. Specify the label to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="label">
        <Radio text="Milky Way" label={<Label>6</Label>} />
    </Story>
</Preview>

### Count

A radio can have a [count](?path=/story/components-count--default-story) after his text. Specify the count to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="count">
        <Radio text="Milky Way" count={<Count>6</Count>} />
    </Story>
</Preview>

### Size

A radio can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Radio size="small" text="Mars" />
            <Radio text="Mars" />
            <Radio size="large" text="Mars" />
        </div>
    </Story>
</Preview>
