# Radio

Radio buttons allow a user to select one option from a set.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Radio", component: InnerRadio }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-checkbox--text">checkbox</a> component.
</WarningMessage>

### Label

A radio can have a label.

<Preview>
    <Story name="label">
        <Radio label="Mars"/>
    </Story>
</Preview>

### Selected

A radio can be selected.

<Preview>
    <Story name="selected">
        <Radio defaultChecked label="Mars"/>
    </Story>
</Preview>

### Disabled

A radio can be disabled.

<Preview>
    <Story name="disabled">
        <Radio disabled label="Mars"/>
    </Story>
</Preview>

### Readonly

A radio can be readonly.

<Preview>
    <Story name="readonly">
        <Radio readOnly label="Mars" />
    </Story>
</Preview>

### No label

A radio can be rendered without a label.

<Preview>
    <Story name="no label">
        <Radio />
    </Story>
</Preview>

### Icons

A radio can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Radio label="Milky Way" icons={<LightbulbIcon />} />
            <Radio label="Milky Way" icons={[<LightbulbIcon />, <InfoIcon />]} />
        </div>
    </Story>
</Preview>

### Badge

A radio can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <div className="flex items-lot-of-space-between">
            <Radio label="Milky Way" badge={<Badge>6</Badge>} />
            <Radio label="Milky Way" badge={<Badge inline>6</Badge>} />
        </div>
    </Story>
</Preview>

### Size

A radio can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Radio size="small" label="Mars" />
            <Radio label="Mars" />
            <Radio size="large" label="Mars" />
        </div>
    </Story>
</Preview>
