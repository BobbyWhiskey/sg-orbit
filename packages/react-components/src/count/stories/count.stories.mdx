# Count

Associate a quantity with a text or a component.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Count", component: PureCount }
    ]}
/>

## Usage

A count component is not meant to be standalone. It's an *addon component* to use in pair with a text or another component that supports count.

### Default

A default count.

<Preview>
    <Story name="default">
        <Count>6</Count>
    </Story>
</Preview>

<Story
    name="knobs"
    decorators={[withKnobs]}
    parameters={paramsBuilder()
        .showPanel()
        .selectedPanel("Knobs")
        .excludeFromDocs()
        .build()}
>
    <Count className={textKnob("className")}>6</Count>
</Story>

### With a text

A count can be pair with a text.

<Preview>
    <Story name="text">
        <span>
            Satellites <Count>6</Count>
        </span>
    </Story>
</Preview>

### Custom color

A quantity color can be customized.

<Preview>
    <Story name="custom color">
        <span>
            Satellites <Count className="primary-500">6</Count>
        </span>
    </Story>
</Preview>

    name="ref"
