# Switch

A switch offers an on or off choice to a user.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Switch", component: InnerSwitch }
    ]}
/>

## Accessibility

This component follow [WAI design pattern for checkbox](https://www.w3.org/TR/wai-aria-practices-1.1/#checkbox).

## Slots

A switch component support the following [slots](?path=/docs/getting-started-slots--page): `"label"`, `"icon"`, `"counter"`.

## Usage

### Label

A default switch.

<Preview>
    <Story name="default">
        <Switch>Engines</Switch>
    </Story>
</Preview>

### Checked

A switch can be checked.

<Preview>
    <Story name="checked">
        <Switch defaultChecked>Engines</Switch>
    </Story>
</Preview>


### No label

A switch can be rendered without a label.

<Preview>
    <Story name="no label">
        <Switch />
    </Story>
</Preview>

### Disabled

A switch can be disabled.

<Preview>
    <Story name="disabled">
        <Switch disabled>Engines</Switch>
    </Story>
</Preview>

### Icon

A switch can have [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icon">
        <Inline>
            <Switch>
                <Text>Pressurization</Text>
                <LightbulbIcon />
            </Switch>
            <Switch>
                <Text>Engines</Text>
                <IconGroup>
                    <LightbulbIcon /><LightbulbIcon />
                </IconGroup>
            </Switch>
        </Inline>
    </Story>
</Preview>

### Counter

A switch can have a [counter](?path=/docs/components-counter--default-story) after his text.

<Preview>
    <Story name="counter">
        <Inline>
            <Switch>
                <Text>Pressurization</Text>
                <Counter>6</Counter>
            </Switch>
            <Switch>
                <Text>Engines</Text>
                <Counter variant="divider">6</Counter>
            </Switch>
        </Inline>
    </Story>
</Preview>

### Size

A switch can vary in size.

<Preview>
    <Story name="size">
        <Inline align="end">
            <Switch size="sm">Engines</Switch>
            <Switch>Engines</Switch>
            <Switch size="lg">Engines</Switch>
        </Inline>
    </Story>
</Preview>

### Reverse

A checkbox checkmark can be to the left or to the right of his label.

<Preview>
    <Story name="reverse">
        <Switch reverse>
            <InfoIcon />
            <Text>Engines</Text>
            <Counter variant="divider">6</Counter>
        </Switch>
    </Story>
</Preview>

### Validation

A switch can display a validation state to communicate to the user whether the current value is valid or invalid. Implement your own validation logic in your app and pass either `"valid"` or `"invalid"` to the input via the `validationState` prop.

The example below illustrates how one would validate if the user has checked the switch.

<FilePreview filePath="/switch/stories/CheckedValidation">
    <CheckedValidation />
</FilePreview>

### Render Props

A switch can update is content with a render props based on the state values.

<Preview>
    <Story name="render props">
        <Switch>
            {
                ({ isChecked }) => {
                    return isChecked ? "On" : "Off"
                }
            }
        </Switch>
    </Story>
</Preview>
