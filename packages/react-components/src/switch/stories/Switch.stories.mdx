# Switch

A switch offers an on or off choice to a user.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Switch", component: InnerSwitch }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-checkbox--text">checkbox</a> component.
</WarningMessage>

### Label

A switch can have a label.

<Preview>
    <Story name="label">
        <Switch label="Orion Arm"/>
    </Story>
</Preview>

### On

A switch can be on.

<Preview>
    <Story name="on">
        <Switch defaultChecked label="Orion Arm"/>
    </Story>
</Preview>

### Disabled

A switch can be disabled.

<Preview>
    <Story name="disabled">
        <Switch disabled label="Orion Arm"/>
    </Story>
</Preview>

### Readonly

A switch can be readonly.

<Preview>
    <Story name="readonly">
        <Switch readOnly label="Orion Arm" />
    </Story>
</Preview>

### No label

A switch can be rendered without a label.

<Preview>
    <Story name="no label">
        <Switch />
    </Story>
</Preview>

### Icons

A switch can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Switch label="Milky Way" icons={<LightbulbIcon />} />
            <Switch label="Milky Way" icons={[<LightbulbIcon />, <InfoIcon />]} />
        </div>
    </Story>
</Preview>

### Badge

A switch can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <div className="flex items-lot-of-space-between">
            <Switch label="Milky Way" badge={<Badge>6</Badge>} />
            <Switch label="Milky Way" badge={<Badge inline>6</Badge>} />
        </div>
    </Story>
</Preview>

### Size

A switch can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Switch size="small" label="Orion Arm" />
            <Switch label="Orion Arm" />
            <Switch size="large" label="Orion Arm" />
        </div>
    </Story>
</Preview>
