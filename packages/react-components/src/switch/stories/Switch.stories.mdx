# Switch

A switch offers an on or off choice to a user.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Switch", component: InnerSwitch }
    ]}
/>

## Usage

### Label

A switch can have a label.

<Preview>
    <Story name="label">
        <Switch>Start engine</Switch>
    </Story>
</Preview>

### On

A switch can be on.

<Preview>
    <Story name="on">
        <Switch defaultOn>Start engine</Switch>
    </Story>
</Preview>


### No label

A switch can be rendered without a label.

<Preview>
    <Story name="no label">
        <Switch />
    </Story>
</Preview>

### Disabled

A switch can be disabled.

<Preview>
    <Story name="disabled">
        <Switch disabled>Start engine</Switch>
    </Story>
</Preview>

### Readonly

A switch can be readonly.

<Preview>
    <Story name="readonly">
        <Switch readOnly>Start engine</Switch>
    </Story>
</Preview>

### Icons

A switch can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icons">
        <Inline>
            <Switch icon={<LightbulbIcon />}>Pressurize cabin</Switch>
            <Switch icon={<IconGroup><LightbulbIcon /><InfoIcon /></IconGroup>}>Start engine</Switch>
        </Inline>
    </Story>
</Preview>

### Badge

A switch can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <Inline>
            <Switch badge={<Badge>6</Badge>}>Pressurize cabin</Switch>
            <Switch badge={<Badge variant="inline">6</Badge>}>Start engine</Switch>
        </Inline>
    </Story>
</Preview>

### Size

A switch can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <Inline align="end">
            <Switch size="small">Start engine</Switch>
            <Switch>Start engine</Switch>
            <Switch size="large">Start engine</Switch>
        </Inline>
    </Story>
</Preview>
