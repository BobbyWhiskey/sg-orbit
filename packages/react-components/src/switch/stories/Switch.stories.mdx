# Switch

A switch offers an on or off choice to a user.

## Installation

## Usage

### Label

A switch can have a label.

<Preview>

    </Story>

## API

To use an [API function](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement#Methods), first get a [ref](https://reactjs.org/docs/refs-and-the-dom.html) on the text switch component, then call the API function.

Here's an example for the `focus` function:

```jsx dark
function MyComponent() {
    const ref = useRef();

    return (
        <div onFocus={() => { ref.focus(); }}>
            <Switch ref={ref} />
        </div>
    );
}
