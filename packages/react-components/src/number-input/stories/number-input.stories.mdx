## API

A few API functions are available for a number input component. To use an API, first get a [ref](https://reactjs.org/docs/refs-and-the-dom.html) on the number input component, then call the API function.

### focus

A number input can be focused programmatically.

```jsx dark
function MyComponent() {
    const ref = useRef();

    return (
        <div onFocus={() => { ref.focus(); }}>
            <NumberInput ref={ref} />
        </div>
    );
}
```

### select

A number input text can be [selected](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement/select) programmatically.

```jsx dark
function MyComponent() {
    const ref = useRef();

    return (
        <NumberInput
            ref={ref}
            onFocus={() => { ref.select(); }}
        />
    );
}
```
