## Usage

### Items

A select component only accepts items as plain objects through the `options` property. You can't specify `<Select.Item />` as subcomponents.

To provide a select item as a shorthand object, you can define an object literal having the following format: `{ text, value, ... }`. To avoid redundant keys, use the `selectItem(text, value, additionalProps?)` function.

For a list of all the properties available on a select item and usage examples, please have a look at the [item page](?path=/docs/components-select-item--text).

```jsx
<Select
    options={[
        selectItem("Julie Payette", "Julie Payette", { disabled: true })
    ]}
/>
```

### Constants

The following constants are used in the examples below:
