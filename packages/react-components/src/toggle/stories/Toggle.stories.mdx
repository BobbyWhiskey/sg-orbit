# Toggle

A toggle offers an on or off choice to a user.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Toggle", component: InnerToggle }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-checkbox--text">checkbox</a> component.
</WarningMessage>

### Text

A toggle can have a text.

<Preview>
    <Story name="text">
        <Toggle text="Orion Arm"/>
    </Story>
</Preview>

### On

A toggle can be on.

<Preview>
    <Story name="on">
        <Toggle defaultChecked text="Orion Arm"/>
    </Story>
</Preview>

### Disabled

A toggle can be disabled.

<Preview>
    <Story name="disabled">
        <Toggle disabled text="Orion Arm"/>
    </Story>
</Preview>

### Readonly

A toggle can be readonly.

<Preview>
    <Story name="readonly">
        <Toggle readOnly text="Orion Arm" />
    </Story>
</Preview>

### No text

A toggle doesn't require a text.

<Preview>
    <Story name="no text">
        <Toggle />
    </Story>
</Preview>

### Icons

A toggle can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text. Specify the icons to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Toggle text="Milky Way" icons={<LightbulbIcon />} />
            <Toggle text="Milky Way" icons={[<LightbulbIcon />, <InfoIcon />]} />
        </div>
    </Story>
</Preview>

### Label

A toggle can have a [label](?path=/docs/components-label--default-story) after his text. Specify the label to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="label">
        <Toggle text="Milky Way" label={<Label>6</Label>} />
    </Story>
</Preview>

### Count

A toggle can have a [count](?path=/story/components-count--default-story) after his text. Specify the count to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="count">
        <Toggle text="Milky Way" count={<Count>6</Count>} />
    </Story>
</Preview>

### Size

A toggle can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Toggle size="small" text="Orion Arm" />
            <Toggle text="Orion Arm" />
            <Toggle size="large" text="Orion Arm" />
        </div>
    </Story>
</Preview>
