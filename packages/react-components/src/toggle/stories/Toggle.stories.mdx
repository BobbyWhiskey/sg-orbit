# Toggle

A toggle offers an on or off choice to a user.

## Installation

<InstallReactComponents imports={<Imports />} />

## Props

<HocWarning componentName="Checkbox" semanticPath="modules/checkbox/" />

<Props
    componentsDefinitions={[
        { displayName: "Toggle", component: InnerToggle }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-checkbox--text">checkbox</a> component.
</WarningMessage>

### Label

A toggle can have a label.

<Preview>
    <Story name="label">
        <Toggle label="Orion Arm"/>
    </Story>
</Preview>

### On

A toggle can be on.

<Preview>
    <Story name="on">
        <Toggle defaultChecked label="Orion Arm"/>
    </Story>
</Preview>

### Disabled

A toggle can be disabled.

<Preview>
    <Story name="disabled">
        <Toggle disabled label="Orion Arm"/>
    </Story>
</Preview>

### Readonly

A toggle can be readonly.

<Preview>
    <Story name="readonly">
        <Toggle readOnly label="Orion Arm" />
    </Story>
</Preview>

### No label

A toggle can be rendered without a label.

<Preview>
    <Story name="no label">
        <Toggle />
    </Story>
</Preview>

### Icons

A toggle can display a single or multiple [icons](?path=/docs/materials-icons--page) after his text.

<Preview>
    <Story name="icons">
        <div className="flex items-lot-of-space-between">
            <Toggle label="Milky Way" icons={<LightbulbIcon />} />
            <Toggle label="Milky Way" icons={[<LightbulbIcon />, <InfoIcon />]} />
        </div>
    </Story>
</Preview>

### Badge

A toggle can have a [badge](?path=/docs/components-badge--default-story) after his text.

<Preview>
    <Story name="badge">
        <div className="flex items-lot-of-space-between">
            <Toggle label="Milky Way" badge={<Badge>6</Badge>} />
            <Toggle label="Milky Way" badge={<Badge inline>6</Badge>} />
        </div>
    </Story>
</Preview>

### Size

A toggle can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-lot-of-space-between">
            <Toggle size="small" label="Orion Arm" />
            <Toggle label="Orion Arm" />
            <Toggle size="large" label="Orion Arm" />
        </div>
    </Story>
</Preview>
