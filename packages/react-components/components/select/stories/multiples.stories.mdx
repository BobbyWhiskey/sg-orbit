# Multiple Values Select

A select can allow a user to make multiple choices.

## Installation

<Installation componentFolder="select" imports={<Imports />} />

## Props

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Select", component: PureSelect }
    ]}
/>

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="?path=/docs/components-select-standard--default-story">standard select</a> component.
</WarningMessage>

### Constants

The following constants are used in the examples below:

export const GENDERS = [
    selectItem("Male", "Male"),
    selectItem("Female", "Female")
];

```javascript dark
const GENDERS = [
    selectItem("Male", "Male"),
    selectItem("Female", "Female")
];
```

### Default

A default multiple values select.

<Preview>
    <Story name="default">
        <Select
            multiple
            options={GENDERS}
        />
    </Story>
</Preview>

### Selected values

A multiple values select can have initial values. _Dont forget to specify values has an array_.

<Preview>
    <Story name="selected values">
        <Select
            placeholder="Gender"
            multiple
            defaultValue={["Female"]}
            options={GENDERS}
        />
    </Story>
</Preview>

### Icon

A multiple values select can display an [icon](?path=/docs/materials-icons--page) before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <Select
            placeholder="Gender"
            multiple
            icon={<MagnifierIcon />}
            options={GENDERS}
        />
    </Story>
</Preview>

### Fluid

A multiple values select can take all the space available.

<Preview>
    <Story name="fluid">
        <Select
            placeholder="Gender"
            multiple
            fluid
            options={GENDERS}
        />
    </Story>
</Preview>

### Disabled
