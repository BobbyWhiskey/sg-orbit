A select can be used to select between choices.

### Constants

The following constants are used in the examples below:

export const MALE = { key: "Male", text: "Male", value: "Male" };
export const FEMALE = { key: "Female", text: "Female", value: "Female" };
export const GENDERS = [
    { key: "Male", text: "Male", value: "Male" },
    { key: "Female", text: "Female", value: "Female" }
];

export const JENNY = { key: "Jenny Hess", text: "Jenny Hess", value: "Jenny Hess" };
export const ELLIOT = { key: "Elliot Fu", text: "Elliot Fu", value: "Elliot Fu" };
export const FRIENDS = [JENNY, ELLIOT];

```javascript dark
const MALE = { key: "Male", text: "Male", value: "Male" };
const FEMALE = { key: "Female", text: "Female", value: "Female" };
const GENDERS = [MALE, FEMALE];

const JENNY = { key: "Jenny Hess", text: "Jenny Hess", value: "Jenny Hess" };
const ELLIOT = { key: "Elliot Fu", text: "Elliot Fu", value: "Elliot Fu" };
const FRIENDS = [JENNY, ELLIOT];
```

### Default

A default select.

<Preview>
    <Story name="default">
        <Select
            options={GENDERS}
        />
    </Story>
</Preview>

### Selected value

A select can have an initial value.

<Preview>
    <Story name="selected value">
        <Select
            placeholder="Gender"
            defaultValue="Female"
            options={GENDERS}
        />
    </Story>
</Preview>

### Icon

A select can display an icon before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <Select
            placeholder="Gender"
            icon={<MagnifierIcon />}
            options={GENDERS}
        />
    </Story>
</Preview>

### Clearable

A clearable select let a user remove is selection.

<Preview>
    <Story name="clearable">

            clearable
            options={GENDERS}
        />
    </Story>
</Preview>

### Size

A select can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Select
                placeholder="Gender"
                fluid
                size="small"
                options={GENDERS}
            />
            <Select
                placeholder="Gender"
                fluid
                options={GENDERS}
            />
            <Select
                placeholder="Gender"
                fluid
                size="large"
                options={GENDERS}
            />
        </div>
    </Story>
</Preview>

A select can show that the data contains errors.

<Preview>
