## Usage

### Items

A select component only accepts items as [object shorthands](?path=/docs/getting-started-shorthand-props--page#object-as-value) through the `options` property. You can't specify `<Select.Item />` as subcomponents.

To provide a select item as a shorthand object, you can define an object literal having the following format: `{ text, value, ... }`. To avoid redundant keys, use the `selectItem(text, value, additionalProps?)` function.

For a list of all the properties available on a select item and usage examples, please have a look at the [item page](?path=/docs/components-select-item--text).

```jsx dark
<Select
    options={[
        selectItem("Male", "Male"),
        selectItem("Female", "Female", { disabled: true })
    ]}
/>
```

### Constants

The following constants are used in the examples below:

export const GENDERS = [
    selectItem("Male", "Male"),
    selectItem("Female", "Female")
];

export const FRIENDS = [
    selectItem("Jenny Hess", "Jenny Hess"),
    selectItem("Elliot Fu", "Elliot Fu")
];

```javascript dark
const GENDERS = [
    selectItem("Male", "Male"),
    selectItem("Female", "Female")
];

const FRIENDS = [
    selectItem("Jenny Hess", "Jenny Hess"),
    selectItem("Elliot Fu", "Elliot Fu")
];
```

A select can display an [icon](?path=/docs/materials-icons--page) before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

### Actions

A select can contains non selectable items acting as actions.

Actions must be specified as [object shorthands](?path=/docs/getting-started-shorthand-props--page#object-as-value) having the following format: `{ content, className? ... }`.

<Preview>
    <Story name="actions">
        <Select
            placeholder="Gender"
            options={GENDERS}
            actions={[{ content: <a href="https://en.wikipedia.org/wiki/Gender" target="_blank" rel="noopener noreferrer">View on Wikipedia</a> }]}
        />
    </Story>
</Preview>
