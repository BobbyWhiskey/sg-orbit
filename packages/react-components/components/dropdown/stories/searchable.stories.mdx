# Searchable Dropdown

A dropdown can allow a user to search through a large list of choices.

## Installation

<Installation componentFolder="dropdown" imports={<Imports />} />

## Usage

<WarningMessage>
    This component have the same feature set as the <a href="/?path=/docs/components-dropdown-standard--default-story">standard dropdown</a> component.
</WarningMessage>

### Default

A default searchable dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            placeholder="Select language"
            selection
            search
            options={LANGUAGES}
        />
    </Story>
</Preview>

### Selected value

A searchable dropdown can have an initial value.

<Preview>
    <Story name="selected value">
        <Dropdown
            placeholder="Select language"
            selection
            search
            defaultValue="Dutch"
            options={LANGUAGES}
        />
    </Story>
</Preview>

### Clearable

A clearable & searchable dropdown let a user remove is selection.

<Preview>
    <Story name="clearable">
        <Dropdown
            placeholder="Select language"
            selection
            search
            clearable
            options={LANGUAGES}
        />
    </Story>
</Preview>

### Fluid

A searchable dropdown can take all the space available.

<Preview>
    <Story name="fluid">
        <Dropdown
            placeholder="Select language"
            selection
            search
            fluid
            options={LANGUAGES}
        />
    </Story>
</Preview>
