# Dropdown

A dropdown can be used to select between choices.

## Usage

### Default

A default dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            selection
            options={GENDERS}
        />
    </Story>
</Preview>

### Selected value

A dropdown can have an initial value.

<Preview>
    <Story name="selected value">
        <Dropdown
            placeholder="Gender"
            selection
            defaultValue="Female"
            options={GENDERS}
        />
    </Story>
</Preview>

### Complex items

A dropdown can render complex items.

<Preview>
    <Story name="complex">
        <Dropdown
            placeholder="Select friend"
            selection
            options={FRIENDS}
        />
    </Story>
</Preview>

### Clearable

A clearable dropdown let a user remove is selection.

<Preview>
    <Story name="clearable">
        <Dropdown
            placeholder="Gender"
            selection
            clearable
            options={GENDERS}
        />
    </Story>
</Preview>

### Custom trigger

A dropdown can have a custom trigger.

<Preview>
    <Story name="custom trigger">
        <Dropdown
            trigger={<AddIcon />}
            options={GENDERS}
        />
    </Story>
</Preview>

### Disabled

A dropdown can be disabled.

<Preview>
    <Story name="disabled">
        <Dropdown
            placeholder="Gender"
            disabled
            selection
            options={GENDERS}
        />
    </Story>
</Preview>

### Fluid

A dropdown can take the width of its container.

<Preview>
    <Story name="fluid">
        <Dropdown
            placeholder="Gender"
            selection
            fluid
            options={GENDERS}
        />
    </Story>
</Preview>

### Size

A dropdown can vary in size.

Can be set to: `"small"`, `"large"` or omitted.
