# Dropdown

A dropdown can be used to select between choices.

## Installation

<Installation componentFolder="dropdown" imports={<Imports />} />

## Usage

<HocWarning componentName="Dropdown" semanticPath="/modules/dropdown" />

<Props
    componentsDefinitions={[
        { displayName: "Dropdown", component: PureDropdown },
        { displayName: "Dropdown.Item", component: PureDropdown.Item }
    ]}
/>

### Constants

The following constants are used in the examples below:

export const MALE = { key: "Male", text: "Male", value: "Male" };
export const FEMALE = { key: "Female", text: "Female", value: "Female" };
export const GENDERS = [
    { key: "Male", text: "Male", value: "Male" },
    { key: "Female", text: "Female", value: "Female" }
];

export const JENNY = { key: "Jenny Hess", text: "Jenny Hess", value: "Jenny Hess" };
export const ELLIOT = { key: "Elliot Fu", text: "Elliot Fu", value: "Elliot Fu" };
export const FRIENDS = [JENNY, ELLIOT];

```javascript dark
const MALE = { key: "Male", text: "Male", value: "Male" };
const FEMALE = { key: "Female", text: "Female", value: "Female" };
const GENDERS = [MALE, FEMALE];

const JENNY = { key: "Jenny Hess", text: "Jenny Hess", value: "Jenny Hess" };
const ELLIOT = { key: "Elliot Fu", text: "Elliot Fu", value: "Elliot Fu" };
const FRIENDS = [JENNY, ELLIOT];
```

### Default

A default dropdown.

<Preview>
    <Story name="default">
        <Dropdown
            selection
            options={GENDERS}
        />
    </Story>
</Preview>

### Selected value

A dropdown can have an initial value.

<Preview>
    <Story name="selected value">
        <Dropdown
            placeholder="Gender"
            selection
            defaultValue="Female"
            options={GENDERS}
        />
    </Story>
</Preview>

### Icon

A dropdown can display an icon before it's content. Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <Dropdown
            placeholder="Gender"
            selection
            icon={<MagnifierIcon />}
            options={GENDERS}
        />
    </Story>
</Preview>

### Clearable

A clearable dropdown let a user remove is selection.

<Preview>
    <Story name="clearable">
        <Dropdown
            placeholder="Gender"
            selection
            clearable
            options={GENDERS}
        />
    </Story>
</Preview>

### Custom trigger

A dropdown can have a custom trigger.

<Preview>
    <Story name="custom trigger">
        <Dropdown
            trigger={<AddIcon />}
            options={GENDERS}
        />
    </Story>
</Preview>

### Disabled

A dropdown can be disabled.

<Preview>
    <Story name="disabled">
        <Dropdown

            disabled
            selection
            options={GENDERS}
        />
    </Story>

### Fluid

A dropdown can take the width of its container.

<Preview>
    <Story name="fluid">
        <Dropdown
            placeholder="Gender"
            selection
            fluid
            options={GENDERS}
        />
    </Story>

### Loading

A dropdown can show that it is currently loading data.

<Preview>
    <Story name="loading">
        <Dropdown
            placeholder="Friends"
            loading
            selection
            options={FRIENDS}
        />
    </Story>
</Preview>

### Size

A dropdown can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Dropdown
                placeholder="Gender"
                selection
                fluid
                size="small"
                options={GENDERS}
            />
            <Dropdown
                placeholder="Gender"
                selection
                fluid
                options={GENDERS}
            />
            <Dropdown
                placeholder="Gender"
                selection
                fluid
                size="large"
                options={GENDERS}
            />
        </div>
    </Story>
</Preview>

### Actions

A dropdown can contains non selectable items acting as actions.

<Preview>
    <Story name="actions">
        <Dropdown
            placeholder="Gender"
            selection
            options={GENDERS}
            actions={[{ content: <a href="https://en.wikipedia.org/wiki/Gender" target="_blank" rel="noopener noreferrer">View on Wikipedia</a> }]}
        />
    </Story>
</Preview>

### Item avatar

A dropdown item can have an avatar.

<Preview>
    <Story name="item avatar">
        <Dropdown
            placeholder="Friends"
            selection
            options={[
                { ...JENNY, avatar: { src: "https://randomuser.me/api/portraits/women/12.jpg", alt: "Jenny" } },
                { ...ELLIOT, avatar: { src: "https://randomuser.me/api/portraits/men/12.jpg", alt: "Elliot" } }
            ]}
        />
    </Story>
</Preview>

### Item icons

A dropdown item can display icons.

<Preview>
    <Story name="item icons">
        <div className="flex items-space-between">
            <Dropdown
                placeholder="Gender"
                selection
                options={[
                    { ...MALE, icons: <WarningIcon /> },
                    { ...FEMALE, icons: [<WarningIcon />, <InfoIcon />] }
                ]}
            />
            <Dropdown
                placeholder="Gender"
                selection
                options={[
                    { ...MALE, icons: <WarningIcon />, iconsPosition: "right" },
                    { ...FEMALE, icons: [<WarningIcon />, <InfoIcon />], iconsPosition: "right" }
                ]}
            />
        </div>
    </Story>
</Preview>
