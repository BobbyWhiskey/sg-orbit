## Usage

### Default

A default button.

<Preview>
    <Story name="default">
        <Button type="button">Cutoff</Button>
    </Story>
</Preview>

### Active

A button can be active.

<Preview>
    <Story name="active">
        <Button type="button" active>Cutoff</Button>
    </Story>
</Preview>

### Compact

A button can reduce its padding to fit into tighter spaces.

<Preview>
    <Story name="compact">
        <Button type="button" compact>Cutoff</Button>
    </Story>
</Preview>

### Disabled

A button can be disabled.

<Preview>
    <Story name="disabled">
        <Button type="button" disabled>Cutoff</Button>
    </Story>
</Preview>

### Fluid

A button can take the width of its container.

<Preview>
    <Story name="fluid">
        <Button type="button" fluid>To infinity and beyond</Button>
    </Story>
</Preview>

### Icon

A button can contain an icon.

<Preview>
    <Story name="icon">
        <div className="flex flex-row">
            <Button type="button" icon><SignoutIcon24 className="w6 h6 mr1" />Click me</Button>
            <Button type="button"><AddIcon className="w6 h6"/></Button>
        </div>
    </Story>
</Preview>

### Label

A button can contain a label.

<Preview>
    <Story name="label">
        <Button type="button" compact label>
            <span className="tl w12">Cutoff</span>
            <Label as="span" size="mini">6</Label>
        </Button>
    </Story>
</Preview>

### Loading

A button can show a loading indicator.

<Preview>
    <Story name="loading">
        <Button type="button" loading>Loading</Button>
    </Story>
</Preview>

### Size

A button can vary in size.

<Preview>
    <Story name="size">
