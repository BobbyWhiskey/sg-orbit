## Usage

### Default

A default button.

<Preview>
    <Story name="default">
        <Button type="button">Cutoff</Button>
    </Story>
</Preview>

### Active

A button can be active.

<Preview>
    <Story name="active">
        <Button type="button" active>Cutoff</Button>
    </Story>
</Preview>

### Compact

A button can reduce its padding to fit into tighter spaces.

<Preview>
    <Story name="compact">
        <Button type="button" compact>Cutoff</Button>
    </Story>
</Preview>

### Disabled

A button can be disabled.

<Preview>
    <Story name="disabled">
        <Button type="button" disabled>Cutoff</Button>
    </Story>
</Preview>

### Fluid

A button can take the width of its container.

<Preview>
    <Story name="fluid">
        <Button type="button" fluid>To infinity and beyond</Button>
    </Story>
</Preview>

### Icon

A button can contain an [icon](?path=/docs/materials-icons--page). Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <div className="flex items-space-between">
            <Button type="button" icon={<SignoutIcon />}>Click me</Button>
            <Button type="button" icon={<SignoutIcon />} iconPosition="right">Click me</Button>
            <Button type="button" icon={<AddIcon />}></Button>
        </div>
    </Story>
</Preview>

### Label

A button can contain a [label](?path=/story/components-label-standard--default-story). Specify the label to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="label">
        <div className="flex items-space-between">
            <Button type="button" label={<Label>6</Label>}>Cutoff</Button>
            <Button type="button" label={{ content: "12", className: "bg-marine-100" }}>Cutoff</Button>
        </div>
    </Story>
</Preview>

### Tag

A button can contain a [tag](?path=/docs/components-label-tag--default-story). Specify the tag to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="tag">
        <div className="flex items-space-between">
            <Button type="button" tag={<Tag className="bg-primary-500" />}>Cutoff</Button>
            <Button type="button" tag={{ className: "bg-sunray-500" }}>Cutoff</Button>
        </div>
    </Story>
</Preview>

### Loading

A button can show a loading indicator.

<Preview>
    <Story name="loading">
        <Button type="button" loading>Loading</Button>
    </Story>
</Preview>

### Size

A button can vary in size.

Can be set to: `"tiny"`, `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Button type="button" size="tiny">Cutoff</Button>
            <Button type="button" size="small">Cutoff</Button>
            <Button type="button">Cutoff</Button>
            <Button type="button" size="large">Cutoff</Button>
        </div>
    </Story>
</Preview>

### Circular

A button can be circular.

<Preview>
    <Story name="circular">
        <div className="flex">
            <div className="flex items-end items-space-between mr4">
                <Button type="button" circular size="tiny">Aa</Button>
                <Button type="button" circular size="small">Aa</Button>
                <Button type="button" circular>Aa</Button>
                <Button type="button" circular size="large">Aa</Button>
            </div>
            <div className="flex items-end items-space-between">
                <Button type="button" icon={<AddIcon />} circular size="tiny"></Button>
                <Button type="button" icon={<AddIcon />} circular size="small"></Button>
                <Button type="button" icon={<AddIcon />} circular></Button>
                <Button type="button" icon={<AddIcon />} circular size="large"></Button>
            </div>
        </div>
    </Story>
</Preview>

### Emphasis

A button can show different levels of emphasis.

<Preview>
    <Story name="emphasis">
        <div className="flex items-space-between">
            <Button type="button" primary>Primary</Button>
            <Button type="button" secondary>Secondary</Button>
        </div>
    </Story>
</Preview>

### Basic

A basic button have a subtle appearance.

<Preview>
    <Story name="basic">
        <div className="flex items-space-between">
            <Button type="button" basic>Default</Button>
            <Button type="button" basic primary>Primary</Button>
            <Button type="button" basic secondary>Secondary</Button>
        </div>
    </Story>
</Preview>

### Ghost

A ghost button is similar to a standard button, taking the same space as a standard button but doesn't have an visible background.

<Preview>
    <Story name="ghost">
        <div className="flex items-space-between">
            <Button type="button" ghost>Default</Button>
            <Button type="button" ghost primary>Primary</Button>
            <Button type="button" ghost secondary>Secondary</Button>
        </div>
    </Story>
</Preview>

### Naked

A button can be colorless. Use this variant if you need to customize the button.

<Preview>
    <Story name="naked">
        <Button type="button" naked>Cutoff</Button>
    </Story>
</Preview>

### Custom color

A naked button can have a custom color.

<Preview>
    <Story name="custom color">
        <Button
            naked
            style={{
                backgroundColor: "#FFF6E7",
                color: "var(--marine-500)",
                boxShadow: "0px 0px 0px 1px #FEE9C3 inset"
            }}
            type="button"
        >
            Cutoff
        </Button>
    </Story>
</Preview>

### Groups

Buttons can exist together as a group.

<Preview>
    <Story name="groups">

            <Button type="button">All</Button>
            <Button type="button">Inactive</Button>
            <Button type="button" active>Active</Button>
        </Button.Group>
    </Story>
</Preview>
