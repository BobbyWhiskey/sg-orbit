# @orbit-ui/react-popup

## Installation

With npm:

```bash
npm install @orbit-ui/react-popup
```

## Usage

```javascript
import { 
    Popup,
    ...
} from "@orbit-ui/react-popup";
```

Or

```javascript
import {
    Popup,
    TOP_LEFT, 
    TOP_RIGHT,
    TOP_CENTER,
    BOTTOM_LEFT,
    BOTTOM_RIGHT,
    BOTTOM_CENTER
} from "@orbit-ui/react-components";
```

Full documentation available at: https://sg-orbit.netlify.com
