# Tag

A tag display a colored dot before the text.

## Installation

<Installation componentFolder="label" imports={<Imports />} />

## Usage

### Default

A default tag.

<Preview>
    <Story name="default">
        <Label tag>
            <Label circular empty size="mini" color="red" />
            Group added
        </Label>
    </Story>
</Preview>

### Button

A tag can contain a button.

<Preview>
    <Story name="button">
        <Label tag button>
            <Label circular empty size="mini" color="red" />
            Group added
            <Button type="button" icon circular size="tiny" ghost>
                <CloseIcon24 />
            </Button>
        </Label>
    </Story>
</Preview>

### Icon

A tag can contain an icon.

<Preview>
    <Story name="icon">
        <Label tag icon={<CommunicationIcon />} iconPosition="right">
            <Label circular empty size="mini" color="red" />
            Group added
        </Label>
    </Story>
</Preview>
