# Tag

A tag is a small colored dot displayed before the text.

## Installation

<Installation componentFolder="label" imports={<Imports />} />

## Usage

### Default

A default tag. A tag color can be specified by setting a `background-color`.

<Preview>
    <Story name="default">
        <Tag className="bg-marine-500" />
    </Story>
</Preview>

### Size

A tag can vary in size.

The available sizes are `mini`, `tiny`, `small`, `medium`, `large`, `big`, `huge` and `massive`.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Tag size="mini" className="bg-marine-500" />
            <Tag size="tiny" className="bg-marine-500" />
            <Tag size="small" className="bg-marine-500" />
            <Tag size="medium" className="bg-marine-500" />
            <Tag size="large" className="bg-marine-500" />
            <Tag size="big" className="bg-marine-500" />
            <Tag size="huge" className="bg-marine-500" />
            <Tag size="massive" className="bg-marine-500" />
        </div>
    </Story>
</Preview>
