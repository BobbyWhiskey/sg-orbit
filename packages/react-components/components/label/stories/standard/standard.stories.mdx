# Label

A label displays content classification.

## Installation

<Installation componentName="label" imports={<Imports />} />

## Usage

### Default

A default label.

<Preview>
    <Story name="default">
        <Label>Notification Sent</Label>
    </Story>
</Preview>

### Button

A label can include a button.

<Preview>
    <Story name="button">
        <Label>
            Group added
            <Button type="button" icon circular size="tiny" className="transparent">
                <svg width="10" height="10" xmlns="http://www.w3.org/2000/svg">
                    <g transform="rotate(45 7.207 4.793)">
                        <rect x=".121" y="5.707" width="12" height="2" rx="1" />
                        <rect x="5" y=".828" width="2" height="12" rx="1" />
                    </g>
                </svg>
            </Button>
        </Label>
    </Story>
</Preview>

### Icon

A label can include an icon.

<Preview>
    <Story name="icon">
        <div className="flex">
            <Label>
                Notification Sent <MagnifierIcon className="h6 w6 fill-marine-500 ml1" />
            </Label>
            <Label>
                <CommunicationIcon className="h6 w6 fill-marine-500 mr1" />
                Notification Sent
                <MagnifierIcon className="h6 w6 fill-marine-500 ml1" />
            </Label>
        </div>
    </Story>
</Preview>

### Basic

A label can have a subtle appearance.

<Preview>
    <Story name="basic">
        <Label basic>Basic</Label>
    </Story>
</Preview>

### Sizes

A label can vary in size.

<Preview>
    <Story name="sizes">
        <div className="flex items-start">
            <Label size="mini">Notification Sent</Label>
            <Label size="tiny">Notification Sent</Label>
            <Label size="small">Notification Sent</Label>
            <Label>Notification Sent</Label>
            <Label size="large">Notification Sent</Label>
        </div>
    </Story>
</Preview>

### Empty

A label can be empty.

<Preview>
    <Story name="empty">
        <Label circular size="tiny" empty color="red" />
    </Story>
</Preview>

### Circular

A label can have a circular form.

<Preview>
    <Story name="circular">

            <Label circular size="tiny">R</Label>
