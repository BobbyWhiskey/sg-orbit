## Usage

### Default

A default input.

<Preview>
    <Story name="default">
        <Input />
    </Story>
</Preview>

### Value

An input can have an value.

<Preview>
    <Story name="value">
        <Input value="Orbit" />
    </Story>
</Preview>

### Focused

An input can show a user is currently interacting with it.

<Preview>
    <Story name="focused">
        <Input focus placeholder="Search..." />
    </Story>
</Preview>

### Disabled

An input can be disabled.

<Preview>
    <Story name="disabled">
        <Input disabled placeholder="Search..." />
    </Story>
</Preview>

### Fluid

An input can take the width of its container.

<Preview>
    <Story name="fluid">
        <Input fluid placeholder="Search..." />
    </Story>
</Preview>

### Icon

An input can be formatted with an [icon](?path=/docs/materials-icons--page).

    <Story name="icon">

</Preview>

### Loading

An input can show that it is currently loading data.

<Preview>
    <Story name="loading">

### Size

An input can vary in size.

The available sizes are: `"small"`, `"medium"` and `"large"`.

<Preview>
    <Story name="size">
        <div className="flex flex-row items-end items-space-between">
            <div className="flex-auto">

            </div>
            <div className="flex-auto">
                <Input fluid icon={<MagnifierIcon />} placeholder="Search..." />
            </div>
            <div className="flex-auto">
                <Input size="large" icon={<MagnifierIcon />} fluid placeholder="Search..." />
            </div>
        </div>
    </Story>
</Preview>

### Error

<Preview>
    <Story name="error">
