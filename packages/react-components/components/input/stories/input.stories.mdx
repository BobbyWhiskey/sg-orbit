## Usage

### Default

A default input.

<Preview>
    <Story name="default">
        <Input />
    </Story>
</Preview>

### Value

An input can have an value.

<Preview>
    <Story name="value">
        <Input value="Orbit" />
    </Story>
</Preview>

### Focused

An input can show a user is currently interacting with it.

<Preview>
    <Story name="focused">
        <Input focus placeholder="Search..." />
    </Story>
</Preview>

### Disabled

An input can be disabled.

<Preview>
    <Story name="disabled">
        <Input disabled placeholder="Search..." />
    </Story>
</Preview>

### Fluid

An input can take the width of its container.

<Preview>
    <Story name="fluid">
        <Input fluid placeholder="Search..." />
    </Story>
</Preview>

### Icon

An input can be formatted with an icon.

<Preview>
    <Story name="icon">
        <div className="flex flex-row">
            <div className="mr2">
                <Input icon="search" placeholder="Search..." />
            </div>
            <div>
                <Input iconPosition="left" icon="search" placeholder="Search..." />
            </div>
        </div>
    </Story>
</Preview>

### Loading

An input can show that it is currently loading data.

<Preview>
    <Story name="loading">
        <div className="flex flex-row">
            <div className="mr2">
                <Input loading placeholder="Search..." />

            <div>

            </div>
        </div>
    </Story>
</Preview>

### Size

An input can vary in size.

The available sizes are: `"tiny"`, `"small"`, `"medium"` and `"large"`.

<Preview>
    <Story name="size">
        <div className="flex flex-row items-end">
            <div className="flex-auto mr2">

            </div>
            <div className="flex-auto mr2">
                <Input size="small" fluid placeholder="Search..." />
            </div>
            <div className="flex-auto mr2">
                <Input fluid placeholder="Search..." />
            </div>
            <div className="flex-auto mr2">
                <Input size="large" fluid placeholder="Search..." />
            </div>
        </div>
    </Story>
</Preview>
