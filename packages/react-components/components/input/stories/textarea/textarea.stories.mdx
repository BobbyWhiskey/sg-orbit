# TextArea

A textarea can be used to allow for extended user input.

## Usage

### Default

A default TextArea.

<Preview>
    <Story name="default">
        <TextArea />
    </Story>
</Preview>

### Value

A textarea can have an value.

<Preview>
    <Story name="value">
        <TextArea value="Orbit" />
    </Story>
</Preview>

### Focused

A textarea can show a user is currently interacting with it.

<Preview>
    <Story name="focused">
        <TextArea focused placeholder="Tell us more" />
    </Story>
</Preview>


### Disabled

A textarea can be disabled.

<Preview>
    <Story name="disabled">
        <TextArea disabled placeholder="Tell us more" />
    </Story>
</Preview>

### Read only

A textarea can be read only.

<Preview>
    <Story name="readonly">
        <TextArea readOnly value="Amazing app." />
    </Story>
</Preview>

### Error

A textarea can show that the data contains errors.

<Preview>
    <Story name="error">
        <TextArea error placeholder="Tell us more" />
    </Story>
</Preview>

### Success

A textarea can show that it has been validated.

<Preview>
    <Story name="success">
        <TextArea success placeholder="Tell us more" />
    </Story>
</Preview>

### Transparent

A textarea can be transparent.

<Preview>
    <Story name="transparent">
        <TextArea transparent placeholder="Tell us more" />
    </Story>
</Preview>

### Min Height

A textarea can have a minimum height.

<Preview>
    <Story name="min height">
        <TextArea placeholder="Tell us more" style={{ minHeight: 100 }}/>
    </Story>
</Preview>

### Rows

A textarea can have a minimum number of rows.

<Preview>
    <Story name="rows">
        <TextArea rows={3} placeholder="Tell us more" />
    </Story>
</Preview>

### Size

A textarea can vary in size.

Available sizes are: `"small"`, `"medium"` and `"large"`.

            </div>
            <div className="flex-auto">
                <TextArea size="large" placeholder="Tell us more..." />

        </div>
    </Story>
</Preview>
