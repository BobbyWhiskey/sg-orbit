## Usage

### Text

A checkbox can have text.

<Preview>
    <Story name="text">
        <Checkbox text="Milky Way" />
    </Story>
</Preview>

### Checked

A checkbox can be checked.

<Preview>
    <Story name="checked">
        <Checkbox defaultChecked text="Milky Way" />
    </Story>
</Preview>

### Indeterminate

A checkbox can be indeterminate.

<Preview>
    <Story name="indeterminate">
        <Checkbox defaultIndeterminate text="Supernova" />
    </Story>
</Preview>

### Disabled

A checkbox can be disabled.

<Preview>
    <Story name="disabled">
        <Checkbox disabled text="Milky Way" />
    </Story>
</Preview>

### Readonly

A checkbox can be readonly.

<Preview>
    <Story name="readonly">
        <Checkbox readOnly text="Milky Way" />
    </Story>
</Preview>

### No text

A checkbox doesn't require to have text.

<Preview>
    <Story name="no text">
        <Checkbox />
    </Story>
</Preview>

### Icon

A checkbox can have an icon after his text.

<Preview>
    <Story name="icon">
        <Checkbox text="Email" icon={<CommunicationIcon />} />
    </Story>
</Preview>

### Label

A checkbox can have a label after his text.

<Preview>
    <Story name="label">
        <Checkbox text="Email" label={<Label>6</Label>} />
    </Story>
</Preview>
