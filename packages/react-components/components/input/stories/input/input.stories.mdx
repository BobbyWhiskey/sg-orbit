## Usage

### Default

A default input.

<Preview>
    <Story name="default">
        <Input />
    </Story>
</Preview>

### Value

An input can have an value.

<Preview>
    <Story name="value">
        <Input defaultValue="Orbit" />
    </Story>
</Preview>

### Focused

An input can show a user is currently interacting with it.

<Preview>
    <Story name="focused">
        <Input focus placeholder="Search..." />
    </Story>
</Preview>

### Disabled

An input can be disabled.

<Preview>
    <Story name="disabled">
        <Input disabled placeholder="Search..." />
    </Story>
</Preview>

### Readonly

An input can be readonly.

<Preview>
    <Story name="readonly">
        <Input readOnly value="Orbit" />
    </Story>
</Preview>

### Fluid

An input can take the width of its container.

<Preview>
    <Story name="fluid">
        <Input fluid placeholder="Search..." />
    </Story>
</Preview>

### Icon

An input can be formatted with an [icon](?path=/docs/materials-icons--page). Specify the icon to use with a [shorthand value](?path=/docs/getting-started-shorthand-props--page).

<Preview>
    <Story name="icon">
        <div className="flex items-space-between">
            <Input icon={<MagnifierIcon />} placeholder="Search..." />
            <Input icon={<MagnifierIcon />} iconPosition="left" placeholder="Search..." />
        </div>
    </Story>
</Preview>

<Story
    name="icon & loading"
    parameters={paramsBuilder()
        .excludeFromDocs()
        .build()}
>
    <Input icon={<MagnifierIcon />} loading />
</Story>

### Loading

An input can show that it is currently loading data.

<Preview>
    <Story name="loading">
        <div className="flex items-space-between">
            <Input loading placeholder="Search..." />
            <Input loading iconPosition="left" placeholder="Search..." />
        </div>
    </Story>
</Preview>

### Size

An input can vary in size.

Can be set to: `"small"`, `"large"` or omitted.

<Preview>
    <Story name="size">
        <div className="flex flex-row items-end items-space-between">
            <div className="flex-auto">
                <Input size="small" icon={<MagnifierIcon />} fluid placeholder="Search..." />
            </div>
            <div className="flex-auto">
                <Input fluid icon={<MagnifierIcon />} placeholder="Search..." />
            </div>
            <div className="flex-auto">
                <Input size="large" icon={<MagnifierIcon />} fluid placeholder="Search..." />
            </div>
        </div>
    </Story>
</Preview>

### Password

An input can be formatted as any [valid input type](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input), e.g. a password input.

<Preview>
    <Story name="password">
        <Input type="password" placeholder="Password" />
    </Story>
</Preview>

### Error

An input field can show that the data contains errors.

<Preview>
    <Story name="error">
        <Input error placeholder="Search..." />
    </Story>
</Preview>

### Transparent

An input can be transparent.

<Preview>
    <Story name="transparent">
        <Input transparent placeholder="Search..." />
    </Story>
</Preview>
