### Naked

A button can be strip of all colors.

<Preview>
    <Story name="naked">
        <Button naked>Naked</Button>
    </Story>
</Preview>

### Custom colors

A button colors can be customized.

<Preview>
    <Story name="custom colors">
        <div className="flex">
            <Button naked style={{ backgroundColor: "#FFF6E7", color: "var(--marine-500)", border: "1px solid #FEE9C3" }}>Department</Button>
            <Button naked style={{ backgroundColor: "#E4F2FF", color: "var(--marine-500)", border: "1px solid #BCE0FF" }}>External project</Button>
            <Button naked style={{ backgroundColor: "#EFF6F6", color: "var(--marine-500)", border: "1px solid #D6E9E9" }}>Important</Button>
        </div>
    </Story>
</Preview>

### Nested label

<Preview>
    <Story name="nested label">

            <div>
                <Button basic>

            <div>
                <Button basic fluid>

            </div>
        </div>
    </Story>
</Preview>
