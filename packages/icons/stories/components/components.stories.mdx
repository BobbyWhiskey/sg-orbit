# Icon

An icon component to render a custom icon (e.g. Azure specific icons) with Orbit standard sizes.

For Orbit official icons, please refer to the [icon gallery](?path=/docs/materials-icons--page#gallery).

## Installation

<Installation />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Icon", component: Icon },
        { displayName: "MultiVariantIcon", component: MultiVariantIcon }
    ]}
/>

## Usage

### Default

A default icon.

<Preview>
    <Story name="default">
        <Icon content={<GearIcon32 />} />
    </Story>
</Preview>

### Size

An icon can vary in size.

Available sizes are: `"tiny"`, `"small"`, `"medium"`, `"large"`, `"big"`, `"huge"` and `"massive"`.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Icon content={<GearIcon32 />} size="tiny" />
            <Icon content={<GearIcon32 />} size="small" />
            <Icon content={<GearIcon32 />} />
            <Icon content={<GearIcon32 />} size="large" />
            <Icon content={<GearIcon32 />} size="big" />
            <Icon content={<GearIcon32 />} size="huge" />
            <Icon content={<GearIcon32 />} size="massive" />
        </div>
    </Story>
</Preview>
