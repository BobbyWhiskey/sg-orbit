# Icon

An icon component allow you to render a custom icon (e.g. Azure specific icons) with Orbit Design standardize sizes. If you are looking for Orbit UI official icons, please have a look at the [icons page](?path=/docs/materials-icons--page#gallery) in the materials section.

## Installation

<Installation />

## Props

<Props
    componentsDefinitions={[
        { displayName: "Icon", component: Icon },
        { displayName: "MultiVariantIcon", component: MultiVariantIcon }
    ]}
/>

## Usage

### Default

A default icon will render as a `medium` sized icon.

<Preview>
    <Story name="default">
        <Icon source={<AzureIcon32 />} />
    </Story>
</Preview>

### Size

An icon can vary in size.

Available sizes are: `"tiny"`, `"small"`, `"medium"`, `"large"`, `"big"`, `"huge"` and `"massive"`.

<Preview>
    <Story name="size">
        <div className="flex items-end items-space-between">
            <Icon source={<AzureIcon32 />} size="tiny" />
            <Icon source={<AzureIcon32 />} size="small" />
            <Icon source={<AzureIcon32 />} />
            <Icon source={<AzureIcon32 />} size="large" />
            <Icon source={<AzureIcon32 />} size="big" />
            <Icon source={<AzureIcon32 />} size="huge" />
            <Icon source={<AzureIcon32 />} size="massive" />
        </div>
    </Story>
</Preview>

### Color

An icon color can be altered by specifying a fill value with [Tachyons fill classes](?path=/docs/materials-colors--page#tachyons):

<Preview>
    <Story name="color">
        <div className="flex items-end items-space-between">
            <Icon source={<AzureIcon32 />} size="huge" className="fill-primary-100" />
            <Icon source={<AzureIcon32 />} size="huge" className="fill-primary-500" />
            <Icon source={<AzureIcon32 />} size="huge" className="fill-primary-900" />
        </div>
    </Story>
</Preview>

If your color is dynamic, you can use the `style` prop and [Orbit UI color variables](?path=/docs/getting-started-foundation--page#colors):

<Preview>
    <Story name="dynamic-color">
        <div className="flex items-end items-space-between">
            <Icon source={<AzureIcon32 />} size="huge" style={{ fill: "var(--sunray-100)" }} />
            <Icon source={<AzureIcon32 />} size="huge" style={{ fill: "var(--sunray-500)" }} />
            <Icon source={<AzureIcon32 />} size="huge" style={{ fill: "var(--sunray-900)" }} />
        </div>
    </Story>
</Preview>

## Multi variants

If your custom icon support multiple variants, you should use the `<MultiVariantIcon />` component. A Multi-variants components will decide  which variant of an icon to render based on the size.
