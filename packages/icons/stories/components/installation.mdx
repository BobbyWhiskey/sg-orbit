<details><summary >install the packages</summary>

Install from [the React bundle](?path=/docs/getting-started-installation--page) or as a standalone component:

```bash dark
npm install @orbit-ui/icons @orbit-ui/tachyons
```
</details>

<details><summary >then import what you need</summary>

```javascript dark
import { Icon, MultiVariantIcon } from "@orbit-ui/icons";
```

</details>
