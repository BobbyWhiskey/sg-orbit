# Icons

Icons are available as [React components](#react) or as [static files](#static-files).

To be included in Orbit an icon must satisfy the following guidelines:

- It should not be specific to a ShareGate brand or any app, e.g. Azure specific icons.

- It must be used as part of an interface, illustrations are not accepted.

- It shouldn't be colored, any icon that has many colors should be in a product codebase.

## React

### Dimensions

Icons comes with default dimensions. You can customize dimensions either with [Tachyons](/?path=/docs/getting-started-tachyons--page) dimensions classes (e.g. `w6 h6` for a 24x24 icon and `w7 h7` for a 32xp32 icon) or with straight CSS:

```jsx dark
<AddIcon24 className="w6 h6" />
```

If you're icon is intended to be used with an Orbit component, chances are you don't need to specify the icon dimensions since most Orbit components already have pre-determined dimensions for their icons.
