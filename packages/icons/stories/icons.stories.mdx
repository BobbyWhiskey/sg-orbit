# Icons

Icons includes their padding and are generally available in two sizes, _24px_ and _32px_.

If an icon is only available in _32px_, he is considered universal.

## Installation

<Installation />

## Usage

### Dimensions

Icons doesn't have a default dimensions to favour reusability.

This means that you must specify dimensions either with [Tachyons](/?path=/docs/introduction-tachyons--page) dimensions classes (e.g. `w6 h6` for a 24x24 icon and `w7 h7` for a 32xp32 icon) or with straight CSS:

```jsx
<AddIcon24 className="w6 h6" />
```
